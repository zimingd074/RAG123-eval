# 「小米生态电商」用户意图体系设计

> 业务定位：面向小米商城与小米生态商品的 RAG 智能客服，覆盖数码、智能家居、健康与生活方式品类。
>
> APP 语义：购物、订单、地址、发票和会员使用「小米商城 APP」；设备添加、智能家居控制和自动化联动使用「米家 APP」。
>
> 政策声明：本项目仅用于学习与评测，保修、退换货、价保、物流、发票和会员等政策内容均为模拟数据，不代表小米官方真实政策；真实权益以小米官方渠道为准。
>
> 意图分层：参考你给的图，采用 **三级一级意图 + 二级意图** 设计
>
