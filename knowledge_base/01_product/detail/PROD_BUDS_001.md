# Redmi Buds 3 商品详情

## 一、商品概览

- 商品名：Redmi Buds 3
- 商品编号：PROD_BUDS_001
- 价格：159 元
- 颜色/款式：白色
- 规格/存储：半入耳式真无线耳机，双耳机 + 充电盒套装

## 二、核心卖点

1. 半入耳佩戴：耳机不深入耳道，适合长时间通勤、网课和轻量办公。
2. 12mm 大尺寸动圈单元：人声和日常影音表现更饱满，适合流行音乐、播客和短视频。
3. 双麦通话降噪：通话时可降低环境噪声干扰，适合室内会议和户外接电话。

## 四、配件兼容

- 标配配件：Redmi Buds 3 左右耳机、充电盒、USB Type-C 充电线、使用说明书
- 兼容配件：
  - 充电器：可使用正规 5V USB 充电器或电脑 USB 口为充电盒充电
  - 手机/平板/电脑：支持标准蓝牙音频连接，MIUI 设备可获得更完整的弹窗和设置体验
  - 保护套：需选择 Redmi Buds 3 充电盒专用尺寸
- 不兼容：
  - Redmi Buds 3 Pro、小米 Buds 4 系列耳塞和充电盒不可混用
  - 无线充电器：Redmi Buds 3 充电盒不支持无线充电

## 五、适用场景

- 推荐场景：
  - 预算在百元价位，需要轻便真无线耳机
  - 不习惯入耳式耳塞压迫感，偏好半入耳佩戴
  - 以通勤听歌、网课、视频会议、短视频为主
- 不建议场景：
  - 地铁、飞机等强噪声环境：建议选择支持主动降噪的 Redmi Buds 3 Pro 或小米 Buds 4 Pro
  - 高强度运动：半入耳结构稳定性弱于入耳式耳机
  - 需要无线充电、双设备连接或空间音频等进阶能力

## 六、包装清单

Redmi Buds 3 左耳机 x1 / Redmi Buds 3 右耳机 x1 / 充电盒 x1 / USB Type-C 充电线 x1 / 使用说明书 x1

## 七、注意事项

- IP54 防护只覆盖耳机本体的日常汗水和轻微泼溅，不适合游泳、淋浴或水洗
- 半入耳耳机隔音能力有限，嘈杂环境下不建议把音量长时间调得过高
- 低延迟、弹窗快连等功能会受手机型号、系统版本和蓝牙环境影响
- 实时价格、库存和促销活动属于动态信息，应由工具查询
