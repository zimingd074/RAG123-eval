# 米家台灯 Pro 商品详情

## 一、商品概览

- 商品名：米家台灯 Pro
- 商品编号：PROD_LIGHT_002
- 型号：MJTD02YL
- 价格：199 元
- 颜色：白色
- 类型：读写作业智能台灯

## 二、核心卖点

1. 高显色照明：显色指数 Ra90，适合阅读、写字、绘画和桌面学习。
2. 亮度与色温可调：支持 2500K-4800K 色温调节，可在阅读、电脑、夜灯等场景中切换。
3. 米家智能控制：支持米家 APP、小爱同学和本地旋钮/按键调节。

## 四、配件兼容

- 标配配件：米家台灯 Pro 主机、电源适配器、使用说明书
- 兼容配件：
  - 米家 APP：用于调节模式、设置场景和升级固件
  - 小爱音箱：可语音控制开关、亮度和色温
  - 电源适配器：建议使用原装 12V 1A 适配器
- 不兼容：
  - 不支持使用普通台灯调光器串接调光
  - 非原装或规格不匹配适配器可能导致亮度异常、闪烁或损坏
  - 不适合潮湿浴室、户外或儿童可拉扯电源线的位置

## 五、适用场景

- 推荐场景：
  - 学生读写作业、成人阅读、桌面办公和绘画
  - 需要根据时间段调节暖光、冷光和亮度
  - 已使用米家 APP 或小爱音箱，希望桌面照明纳入智能场景
- 不建议场景：
  - 作为房间主照明或大面积补光
  - 对屏幕摄影、专业色彩校准有严格要求的工作室
  - 桌面空间极小或无法稳定摆放台灯的环境

## 六、包装清单

米家台灯 Pro x1 / 电源适配器 x1 / 使用说明书 x1

## 七、注意事项

- 使用时应避免灯头直射眼睛，建议调整到照亮桌面而非正对视线
- 长时间阅读仍应定时休息，台灯不能替代良好坐姿和环境照明
- 仅支持 2.4GHz WiFi 配网，不支持 5GHz；双频合一路由器可参考 [NET_GUIDE_002] 5G/2.4G WiFi 配网说明
- 台灯通过转轴调节角度时不要过度用力，避免损伤机械结构
- 实时价格、库存和促销活动属于动态信息，应由工具查询
