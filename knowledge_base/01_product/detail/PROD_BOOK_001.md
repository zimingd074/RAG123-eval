# RedmiBook Pro 14 商品详情

## 一、商品概览

- 商品名：Redmi Book Pro 14 2024（知识库简称 RedmiBook Pro 14）
- 商品编号：PROD_BOOK_001
- 价格：4999 元起
- 颜色/款式：星辰灰等，具体以在售版本为准
- 规格/存储：Core Ultra 5 / Core Ultra 7，16GB 或 32GB 内存，512GB 或 1TB SSD 等版本

## 二、核心卖点

1. Intel Core Ultra 平台：集成 NPU 和 Intel Arc 核显，适合办公、编程、轻度创作和多任务。
2. 14 英寸 2.8K 120Hz 屏：2880 x 1800 分辨率、16:10 比例和高刷新率，兼顾清晰度与便携。
3. 80Wh 大电池：在 14 英寸轻薄本中电池容量较高，适合长时间移动办公。

## 四、配件兼容

- 标配配件：Redmi Book Pro 14 2024 主机、100W USB-C 电源适配器、电源线、快速入门指南、保修卡
- 兼容配件：
  - 充电器：支持对应功率 USB-C PD 的正规充电器，原装 100W 更稳妥
  - 外接显示器：可通过 Thunderbolt 4 / USB-C 或 HDMI 2.1 输出，显示规格受线材和显示器限制
  - 扩展坞：建议选择支持 USB-C PD、HDMI 或 DP 输出的合规扩展坞
- 不兼容：
  - 不建议使用低功率手机充电器作为长期供电，可能无法满足高负载功耗
  - 游戏本大功率方口适配器不能直接用于该机 USB-C 充电口

## 五、适用场景

- 推荐场景：
  - 学生论文、轻度编程、Office 办公、网页多开和远程会议
  - 经常携带电脑通勤，希望兼顾高分屏和长续航
  - 不需要独立显卡，但希望核显能覆盖轻度图片处理和视频剪辑
- 不建议场景：
  - 3A 游戏、CUDA 加速、重度建模渲染：建议选择 Redmi G Pro 游戏本
  - 长期插多块外接硬盘或专业采集设备：需额外核对接口带宽和扩展坞规格
  - 对数字小键盘有强需求：14 英寸机身键盘布局不适合该需求

## 六、包装清单

Redmi Book Pro 14 2024 主机 x1 / 100W USB-C 电源适配器 x1 / 电源线 x1 / 快速入门指南 x1 / 保修卡 x1

## 七、注意事项

- 高分辨率和高刷新率会增加耗电，外出办公可按需降低刷新率
- Thunderbolt 4、USB-C 充电和显示输出需要合规线材与外设配合
- Windows 初次激活和系统更新建议连接稳定网络并保持电量充足
- 轻薄本散热空间有限，长时间高负载时应放在平整桌面使用
- 价格活动、库存和具体配置组合属于动态信息，应由工具查询
