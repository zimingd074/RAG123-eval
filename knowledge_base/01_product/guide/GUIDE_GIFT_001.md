# 父母生日礼物推荐指南

## 一、场景描述

给父母选生日礼物，应优先考虑实用、易上手、维护简单和售后清晰。过于复杂的数码功能未必能提升体验，反而可能增加学习成本。适合父母的礼物通常集中在健康生活、清洁、空气质量、出行和基础智能家居几类。

本指南只推荐知识库已有商品，并避免把健康类产品描述成医疗器械。体脂、按摩、口腔护理类数据仅供日常管理和放松参考，不能替代医生诊断。

## 二、选购维度

| 维度 | 说明 | 权重 |
|---|---|---|
| 易用性 | 父母礼物应少设置、少维护、上手直接 | 高 |
| 安全边界 | 健康类和电器类产品必须写清不适用人群和使用限制 | 高 |
| 实用频率 | 高频日用品比新奇小玩具更稳妥 | 高 |
| 维护成本 | 刷头、滤芯、刀头等耗材要提前说明 | 中 |
| 空间适配 | 净化器、扫地机、旅行箱需考虑家中空间和使用习惯 | 中 |
| 预算 | 父母礼物可按 100、300、1000、4000 元分层 | 中 |

## 四、选购避坑

- 不要把体脂秤、按摩仪、牙刷当成医疗设备。慢病、术后、植入医疗设备等情况应先咨询医生。
- 不要给不熟悉智能设备的父母买复杂联动方案。先选单品，再逐步配置 APP。
- 扫地机适合减轻家务，但家中地面杂物多、门槛高时需要先整理环境。
- 净化器滤芯需要定期更换，送礼时应说明耗材成本。
- 剃须刀、牙刷刷头属于个人护理耗材，不建议多人共用。

## 五、总结建议

预算 100-300 元，可选米家体脂秤 S400、米家声波电动牙刷 T500、米家电动剃须刀 S500。预算 800-1000 元，可选米家空气净化器 4。预算更高且希望明显减轻家务，可选石头扫地机器人 G10。父母经常出行时，米家旅行箱 20寸比复杂电子产品更稳妥。
