# 出租屋智能化改造指南

## 一、场景描述

出租屋智能化改造的核心原则是低成本、可搬走、少安装、不破坏原装修。优先选择插电即用、蓝牙或 WiFi 配网、无需打孔或强电改造的设备。涉及门锁、吸顶灯、摄像设备时，应先获得房东同意，并确认退租时能否恢复原状。

本指南推荐知识库中已完成的智能插座、小爱音箱、台灯、空气净化器、摄像机、蓝牙音箱和移动电源等可迁移设备。对于需要施工的智能门锁和吸顶灯，只作为有明确许可时的进阶选项。

## 二、选购维度

| 维度 | 说明 | 权重 |
|---|---|---|
| 可迁移性 | 退租时能带走、能恢复原状的设备更适合出租屋 | 高 |
| 安装难度 | 优先免打孔、免布线、插电即用 | 高 |
| 网络要求 | 多数米家设备需要 2.4GHz WiFi，出租屋路由器需可配置 | 高 |
| 成本控制 | 出租屋改造不宜重资产，优先百元级设备 | 中 |
| 隐私合规 | 摄像机、门铃需考虑室友、邻里和公共区域规则 | 中 |
| 场景价值 | 睡前关灯、定时风扇、空气净化、桌面照明最实用 | 中 |

## 四、选购避坑

- 不建议未经许可更换智能门锁。门体、物业、房东和退租恢复都会带来风险。
- 不建议优先改吸顶灯。吸顶灯涉及市电和安装，出租屋更适合先用台灯、插座和小夜灯。
- 摄像机不能对准公共区域、室友空间或邻居门口，避免隐私争议。
- 出租屋网络常见双频合一或隔离网络，配网前应确认 2.4GHz WiFi 可用。
- 智能插座不应用于高功率电暖器和热水器等无人看管设备。

## 五、总结建议

出租屋首选“米家智能插座 3 + 小爱音箱 Play 增强版 + 米家台灯 Pro”，能覆盖语音控制、定时开关和桌面照明。空气质量差或空间较封闭时补充米家空气净化器 4 Lite。独居用户如需安防，可在合规前提下选择小米智能摄像机云台版 2K。
