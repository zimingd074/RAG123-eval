# 全屋智能家居入门搭配（5000 预算）

## 一、场景描述

5000 元预算做全屋智能，目标不应是一次性买满所有设备，而是先覆盖高频空间和高频动作：入户、客厅、卧室、环境净化和基础安防。入门搭配应优先选择米家生态内已接入的设备，保证米家 APP、小爱音箱和自动化场景能统一管理。

本指南只推荐知识库中已有商品：小爱音箱 Pro、小米智能门锁 E10、米家智能吸顶灯、米家智能插座 3、米家空气净化器 4、小米智能摄像机云台版 2K、小米智能门铃 3 等。实际价格和库存属于动态信息，应由工具查询。

## 二、选购维度

| 维度 | 说明 | 权重 |
|---|---|---|
| 生态兼容 | 优先选择米家 APP 和小爱同学可统一控制的设备 | 高 |
| 高频场景 | 入户开灯、语音控制、空气净化、安防提醒优先级更高 | 高 |
| 安装成本 | 门锁、吸顶灯涉及安装，应预留安装和适配确认时间 | 高 |
| 预算占比 | 门锁和空气净化器占预算较高，插座和音箱适合补齐场景 | 中 |
| 隐私安全 | 摄像机、门铃、门锁涉及家庭隐私和权限管理 | 中 |
| 后续扩展 | 先买中枢和基础设备，后续再加传感器、灯具和家电 | 中 |

## 四、选购避坑

- 不要先买太多传感器。没有音箱、灯、插座等执行设备时，传感器只能提醒，无法形成有效场景。
- 不要忽略安装。智能门锁和吸顶灯涉及门体、市电和安装条件，购买前应确认门体结构和天花板接线。
- 不要把摄像机装在敏感区域。卧室、卫生间等隐私空间不适合安装摄像设备。
- 不要用智能插座控制高功率电器。电暖器、热水器等高负载设备应先核对功率和安全边界。
- 不要把实时价格写死。知识库价格是参考价，活动价、库存和发货时效应走工具查询。

## 五、总结建议

5000 元预算可优先选择“小爱音箱 Pro + 小米智能门锁 E10 + 米家智能吸顶灯 + 米家智能插座 3 + 米家空气净化器 4”，再根据隐私接受度补充小米智能摄像机云台版 2K 或小米智能门铃 3。入门阶段的目标是稳定、可控、可扩展，而不是一次性堆满所有设备。
