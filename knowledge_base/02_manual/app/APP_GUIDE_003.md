# 比特APP 数据查看与导出

## 一、功能入口

- 设备数据入口：比特APP > 首页 > 选择设备 > 数据/记录/历史
- 空气质量：空气净化器设备页 > 空气质量趋势
- 体脂数据：体脂秤设备页 > 身体数据 > 历史记录
- 门锁记录：门锁设备页 > 开锁记录
- 摄像机记录：摄像机设备页 > 回看/事件
- 导出入口：设备数据页 > 右上角“...” > 导出/分享，具体以设备支持为准

## 二、操作步骤

### Step 1：进入设备数据页

1. 打开比特APP。
2. 在首页选择目标设备。
3. 点击“数据”“记录”“历史”或“统计”入口。
4. 按日、周、月或自定义时间查看趋势。

### Step 2：筛选数据范围

1. 选择时间范围。
2. 选择家庭成员或设备子项。
3. 对门锁、摄像机等设备，可按事件类型筛选。
4. 确认页面展示的数据是否为目标设备。

### Step 3：导出或分享

1. 在数据页面点击右上角“...”。
2. 若设备支持导出，选择“导出数据”。
3. 选择文件格式、时间范围和保存位置。
4. 若只支持截图或分享，按页面提示生成图片或链接。

### Step 4：清理历史记录

1. 进入设备设置或数据管理。
2. 查找“清除记录”“删除历史”或“隐私管理”。
3. 删除前确认是否已经备份。
4. 安防类记录删除后可能无法恢复。

## 三、注意事项

- 并非所有设备都支持数据导出；部分设备只支持查看或截图分享。
- 健康类数据如体脂、心率、刷牙记录仅供日常参考，不作为诊断依据。
- 摄像机、门铃和门锁记录涉及隐私，不应随意分享给非家庭成员。
- 云存储录像、历史趋势和高级统计可能受服务套餐影响。
