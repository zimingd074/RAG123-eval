# 跨设备联动配置（小爱音箱控制/设备互动）

## 一、准备工作

跨设备联动用于让小爱音箱、门锁、灯具、插座、净化器、摄像机和扫地机协同工作。所有设备应添加到同一个米家家庭，并按房间命名。语音控制尤其依赖清晰设备名称。

准备设备示例：

1. 语音入口：小爱音箱 Play 增强版或小爱音箱 Pro。
2. 执行设备：米家智能吸顶灯、米家智能插座 3、米家空气净化器 4 系列。
3. 清洁设备：石头扫地机器人 T7 / T7S Plus / G10 / G10S Pro。
4. 安防设备：小米智能门锁、小米智能摄像机、小米智能门铃。

## 二、配网/联动步骤

### Step 1：统一设备名称

1. 打开米家 APP。
2. 进入每个设备的设置页。
3. 将设备改名为“房间 + 设备”，例如“卧室台灯”“客厅净化器”。
4. 避免使用“灯1”“插座2”等难以语音识别的名称。

### Step 2：测试小爱音箱控制

1. 确认小爱音箱在线。
2. 对音箱说“小爱同学，打开卧室台灯”。
3. 测试“关闭客厅插座”“打开空气净化器自动模式”等指令。
4. 控制失败时，检查设备是否在线和名称是否重复。

### Step 3：创建设备互动

1. 进入米家 APP > 智能 > 自动化。
2. 选择触发条件，如“门锁开门”“空气质量变差”“到达指定时间”。
3. 选择执行动作，如“打开灯”“播放提示音”“启动净化器”。
4. 保存后手动运行测试。

### Step 4：设置语音场景

1. 在自动化中创建手动场景。
2. 命名为“观影模式”“睡眠模式”“清洁模式”等。
3. 添加多个执行动作。
4. 对小爱音箱说“小爱同学，执行观影模式”。

## 三、常见失败原因与解决

| 现象 | 原因 | 解决方法 |
|---|---|---|
| 小爱听懂但不执行 | 设备离线 | 在米家 APP 中检查目标设备在线状态 |
| 控制了错误设备 | 名称重复或房间不清晰 | 改为“客厅吸顶灯”“卧室台灯” |
| 场景执行一半 | 某个设备离线或动作不支持 | 删除失败动作或调整执行顺序 |
| 扫地机无法按房间清扫 | 地图未保存或房间未命名 | 在扫地机 App 中保存地图和房间 |
| 红外设备控制失败 | 音箱与家电之间有遮挡 | 调整音箱位置并重新匹配遥控器 |
| 自动化误触发 | 条件过宽 | 添加时间段、设备状态或手动确认 |

## 四、适用设备列表

| 设备 | 联动能力 | 说明 |
|---|---|---|
| 小爱音箱 Play 增强版 | 语音入口、红外遥控 | 适合卧室和出租屋 |
| 小爱音箱 Pro | 语音入口、红外遥控、米家控制 | 适合客厅和全屋入口 |
| 米家智能吸顶灯 | 开关、亮度、色温 | 适合回家和睡眠场景 |
| 米家智能插座 3 | 通断电、定时 | 适合普通小电器 |
| 米家空气净化器 4 系列 | 开关、模式、风量 | 可按时间或空气质量联动 |
| 石头扫地机器人系列 | 清扫、回充、区域清扫 | 具体能力以 App 地图和机型为准 |
| 小米智能门锁系列 | 开锁、上锁、异常提醒 | 安防场景需谨慎授权 |

## 五、补充说明

- 跨设备联动依赖网络、账号、家庭权限和设备在线状态。
- 涉及门锁、摄像机和门铃的联动应注意隐私和权限。
- 配网问题详见 [NET_GUIDE_001]，设备共享详见 [APP_GUIDE_002]。
