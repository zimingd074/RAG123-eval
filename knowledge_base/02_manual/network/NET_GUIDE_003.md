# 路由器更换后设备重连指南

## 一、准备工作

更换路由器、修改 WiFi 名称、修改 WiFi 密码或更换宽带后，米家设备可能显示离线。若新路由器使用与旧路由器完全相同的 2.4GHz WiFi 名称和密码，部分设备可自动重连；若名称或密码变化，大多数设备需要重新配网。

准备事项：

1. 新路由器已联网。
2. 2.4GHz WiFi 已开启。
3. 手机连接到新路由器 WiFi。
4. 米家 APP 已登录原设备账号。
5. 确认每台设备可接触到重置键或电源。

## 二、配网/联动步骤

### Step 1：尝试保留原 WiFi 信息

1. 在新路由器中设置与旧路由器相同的 WiFi 名称。
2. 设置相同密码和加密方式。
3. 重启智能设备。
4. 等待 3-5 分钟，观察米家 APP 是否恢复在线。

### Step 2：重新配网离线设备

1. 在米家 APP 中进入离线设备页。
2. 查看是否有“重新连接 WiFi”入口。
3. 若无入口，按设备说明重置 WiFi。
4. 通过“首页 > + > 添加设备”重新添加。
5. 选择新路由器 2.4GHz WiFi 并输入密码。

### Step 3：检查家庭和房间

1. 设备恢复在线后，确认设备仍在原家庭。
2. 检查房间、设备名称和共享权限是否保留。
3. 测试小爱音箱语音控制是否仍能识别设备名称。
4. 若自动化失效，打开自动化逐条检查条件和执行设备。

### Step 4：处理无法重连设备

1. 删除米家 APP 中的离线设备。
2. 让设备恢复出厂或重置网络。
3. 重新添加设备。
4. 若设备提示已被绑定，需用原账号解除绑定。

## 三、常见失败原因与解决

| 现象 | 原因 | 解决方法 |
|---|---|---|
| 旧设备全部离线 | WiFi 名称或密码变化 | 改回旧名称密码，或逐台重新配网 |
| 部分设备在线部分离线 | 设备只支持 2.4GHz | 确认新路由器 2.4GHz 已开启 |
| 小爱能联网但插座离线 | 音箱支持网络更多，插座只支持 2.4GHz | 为插座单独配置 2.4GHz |
| 自动化不执行 | 设备 ID 或房间变化 | 重新编辑自动化条件和执行动作 |
| 设备提示已绑定 | 未从旧账号删除 | 用原账号解除绑定后重新添加 |
| 路由器后台看不到设备 | MAC 过滤或黑名单 | 关闭接入限制并重启设备 |

## 四、适用设备列表

| 设备 | 重连方式 | 说明 |
|---|---|---|
| 米家智能插座 3 | 重新配网 | WiFi 名称密码变化后通常需重置 |
| 米家空气净化器 4 系列 | 重新配网 | 设备页或机身按键进入配网状态 |
| 小米智能摄像机云台版 2K | 重新配网 | 需保持 2.4GHz 信号稳定 |
| 小爱音箱系列 | APP 重新配置网络 | 配网失败时检查音箱是否静音或未进入配网 |
| 石头扫地机器人系列 | 米家/Roborock APP 重置 WiFi | 需要按设备组合键进入配网状态 |
| 小米智能门锁系列 | 视型号和网关而定 | 蓝牙近场功能不一定受 WiFi 影响 |

## 五、补充说明

- 若家中设备很多，优先保留旧 WiFi 名称和密码，迁移成本最低。
- 路由器双频合一导致失败时，详见 [NET_GUIDE_002]。
- 设备重新添加和共享规则详见 [APP_GUIDE_002]。
