# 智能设备 WiFi 配网通用指南

## 一、准备工作

配网前应准备手机、米家 APP、家庭 WiFi、待添加设备和设备说明书。大多数米家智能设备优先使用 2.4GHz WiFi，部分音箱、手机和平板可使用 5GHz，但扫地机、空气净化器、摄像机、智能插座等设备通常需要 2.4GHz。

配网前检查：

1. 手机已登录米家 APP。
2. 手机蓝牙、定位、本地网络权限已开启。
3. 路由器 2.4GHz WiFi 已开启。
4. 设备已接通电源或电量充足。
5. 手机、设备和路由器距离较近。

## 二、配网/联动步骤

### Step 1：让设备进入配网状态

1. 按设备说明长按重置键或组合键。
2. 等待指示灯闪烁、语音提示或 APP 提示设备待连接。
3. 若设备已绑定旧账号，应先在旧账号中解除绑定。

### Step 2：在米家 APP 添加设备

1. 打开米家 APP 首页。
2. 点击右上角“+”。
3. 选择“添加设备”。
4. 等待 APP 自动发现设备，或手动搜索设备型号。
5. 选择设备后进入配网页面。

### Step 3：选择 WiFi 并输入密码

1. 选择家庭 2.4GHz WiFi。
2. 输入 WiFi 密码。
3. 确认手机没有连接移动热点、访客网络或公司网络。
4. 等待设备连接路由器并绑定账号。

### Step 4：完成后测试

1. 给设备命名并选择房间。
2. 进入设备页测试开关、模式、状态刷新等基础功能。
3. 若需要语音控制，确认设备名称清晰且小爱音箱在线。
4. 若设备显示离线，重启设备和路由器后观察。

## 三、常见失败原因与解决

| 现象 | 原因 | 解决方法 |
|---|---|---|
| 找不到设备 | 设备未进入配网模式 | 按说明重置 WiFi 后重新添加 |
| APP 提示网络不支持 | 手机连接 5GHz WiFi | 切换到 2.4GHz WiFi，详见 [NET_GUIDE_002] |
| 连接到 99% 失败 | 路由器限制新设备 | 关闭 MAC 过滤、AP 隔离、黑名单 |
| 密码输入后失败 | WiFi 密码错误 | 重新输入密码，避免特殊字符误输 |
| 绑定失败 | 设备已绑定其他账号 | 使用旧账号删除设备后重新绑定 |
| 添加后离线 | 信号弱或路由器距离远 | 靠近路由器或增加 Mesh/中继 |
| 公司/校园网失败 | 需要网页认证或企业认证 | 使用家庭路由器或 2.4GHz 手机热点测试 |

## 四、适用设备列表

| 设备 | 配网要求 | 说明 |
|---|---|---|
| 米家空气净化器 4 系列 | 2.4GHz WiFi | 配网时需打开蓝牙和定位 |
| 小米智能摄像机云台版 2K | 2.4GHz WiFi | 需要稳定上传网络 |
| 小米智能门铃 3 | 米家 APP / Xiaomi Home | 需考虑门外信号强度 |
| 米家智能插座 3 | 2.4GHz WiFi | 不适合企业认证网络 |
| 小爱音箱系列 | 2.4GHz WiFi，部分型号支持更多网络 | 以具体型号为准 |
| 石头扫地机器人系列 | 2.4GHz WiFi | 也可按 Roborock APP 提示配网 |
