# 5G/2.4G WiFi 配网说明（避坑）

## 一、准备工作

大多数小米/米家/石头智能家居设备只支持 2.4GHz WiFi 配网，不支持连接 5GHz WiFi。石头 T7、T7S Plus、G10、G10S Pro 的商品详情均标注为 2.4GHz WiFi。

配网前确认：

1. 手机已安装米家 APP 或 Roborock APP。
2. 手机蓝牙、定位权限已开启。
3. 路由器已开启 2.4GHz WiFi。
4. WiFi 密码不包含容易输入错误的特殊字符。
5. 设备、手机和路由器距离较近，避免隔墙太多。

## 二、配网/联动步骤

### Step 1：区分 2.4GHz 和 5GHz WiFi

1. 打开手机 WiFi 列表。
2. 查看家中是否有两个相似名称，例如 Home_2.4G 和 Home_5G。
3. 若有明确 2.4G 名称，选择 2.4G 网络。
4. 若只有一个 WiFi 名称，进入路由器后台确认是否启用双频合一。

### Step 2：关闭或调整双频合一

1. 登录路由器管理页面或路由器 APP。
2. 找到无线设置。
3. 若开启双频合一，临时关闭并分别命名 2.4GHz 和 5GHz。
4. 使用 2.4GHz 网络完成设备配网。
5. 配网成功后，可按路由器兼容性决定是否重新开启双频合一。

### Step 3：重置设备 WiFi

1. 按设备说明进入 WiFi 重置状态。
2. 等待设备语音或指示灯提示进入配网模式。
3. 在米家 APP 或 Roborock APP 中添加设备。
4. 选择 2.4GHz WiFi 并输入密码。
5. 等待设备绑定到账号。

### Step 4：配网后检查

1. 在 APP 首页确认设备在线。
2. 点击设备进入控制页。
3. 测试启动、暂停、回充等基础指令。
4. 若设备显示离线，将设备和路由器距离拉近后重试。

## 三、常见失败原因与解决

| 现象 | 原因 | 解决方法 |
|---|---|---|
| APP 提示不支持当前 WiFi | 手机连接了 5GHz WiFi | 切换到 2.4GHz WiFi 后重新配网 |
| 设备一直连接中 | 双频合一导致设备连到 5GHz | 临时关闭双频合一，拆分 2.4G/5G 名称 |
| 输入密码后失败 | WiFi 密码错误或包含特殊字符 | 重新输入密码，必要时临时改为字母数字组合 |
| 找不到设备 | 蓝牙或定位权限未开启 | 开启手机蓝牙、定位和 APP 权限 |
| 设备绑定后离线 | 路由器信号弱或隔墙太多 | 将设备靠近路由器，或增加 Mesh/中继 |
| 路由器后台看不到设备 | 路由器限制新设备接入 | 关闭 MAC 过滤、黑名单、AP 隔离 |
| 公司/校园网无法配网 | 需要网页登录认证或企业级认证 | 使用家庭路由器或手机热点测试 |
| 热点配网失败 | 热点频段为 5GHz | 将手机热点设置为 2.4GHz 频段 |

## 五、避坑建议

- 给 2.4GHz 和 5GHz 设置不同名称，能减少大量配网失败。
- 配网时手机必须连接目标 2.4GHz WiFi，而不是移动数据或 5GHz WiFi。
- 路由器不要开启 AP 隔离，否则手机可能无法发现局域网设备。
- Mesh 路由环境中，首次配网可临时靠近主路由。
- 更换路由器或 WiFi 名称后，设备通常需要重新配网；详见 [NET_GUIDE_003] 路由器更换后设备重连指南。
