# 智能场景配置指南（回家/离家模式）

## 一、准备工作

回家/离家模式适合把门锁、灯、插座、空气净化器、摄像机、小爱音箱等设备组合成自动化。配置前应确认设备已添加到同一个米家家庭，并且设备名称清晰。

准备设备示例：

1. 入户触发设备：小米智能门锁 E10 或小米全自动智能门锁 Pro。
2. 执行动作设备：米家智能吸顶灯、米家智能插座 3、米家空气净化器 4 系列。
3. 提醒设备：小爱音箱 Pro 或小爱音箱 Play 增强版。
4. 安防设备：小米智能摄像机云台版 2K 或小米智能门铃 3。

## 二、配网/联动步骤

### Step 1：创建回家模式

1. 打开米家 APP。
2. 进入“智能 > 自动化”。
3. 点击“+”创建自动化。
4. 条件选择“门锁开锁”或“指定家庭成员开锁”。
5. 执行动作选择“打开玄关灯”“打开客厅灯”“关闭摄像机隐私模式”等。
6. 保存并命名为“回家模式”。

### Step 2：创建离家模式

1. 进入“智能 > 自动化”。
2. 点击“+”创建自动化。
3. 条件选择“门锁上锁”“手动点击离家场景”或“小爱音箱语音触发”。
4. 执行动作选择“关闭灯光”“关闭插座”“开启摄像机移动侦测”“开启净化器自动模式”。
5. 保存并命名为“离家模式”。

### Step 3：设置生效时间

1. 在自动化条件中添加时间段。
2. 夜间回家可设置低亮度灯光。
3. 白天离家可只关闭插座，不触发安防提示音。
4. 保存后在自动化列表中确认开关已启用。

### Step 4：测试场景

1. 手动点击自动化运行按钮。
2. 检查每个设备是否按预期执行。
3. 若设备未响应，进入设备页确认在线状态。
4. 根据实际体验调整设备名称、时间段和执行顺序。

## 三、常见失败原因与解决

| 现象 | 原因 | 解决方法 |
|---|---|---|
| 自动化不触发 | 条件设备离线或未授权 | 检查门锁、音箱和传感设备在线状态 |
| 灯开错房间 | 设备命名重复 | 修改为“卧室台灯”“客厅吸顶灯”等明确名称 |
| 离家后插座没关 | 插座离线或动作未保存 | 测试插座控制并重新保存自动化 |
| 摄像机误报太多 | 移动侦测灵敏度高 | 调低灵敏度或设置生效时间段 |
| 小爱语音不执行 | 场景名称难识别 | 使用短名称，如“回家模式”“离家模式” |

## 四、适用设备列表

| 设备 | 可用角色 | 说明 |
|---|---|---|
| 小米智能门锁 E10 | 回家/离家触发 | 指纹、密码、上锁可作为条件 |
| 小米全自动智能门锁 Pro | 回家/离家触发 | 可结合可视门铃提醒 |
| 米家智能吸顶灯 | 执行动作 | 回家开灯、夜间低亮 |
| 米家智能插座 3 | 执行动作 | 普通灯、风扇、小电器通断 |
| 米家空气净化器 4 系列 | 执行动作 | 回家前净化或离家自动模式 |
| 小米智能摄像机云台版 2K | 安防动作 | 离家开启移动侦测 |
| 小爱音箱系列 | 语音触发和播报 | 可手动触发场景 |

## 五、补充说明

- 涉及安防和门锁的自动化应先小范围测试，避免误触发。
- 高功率电器不建议通过智能插座无人自动开启。
- 设备共享和权限管理详见 [APP_GUIDE_002]。
