# 小爱音箱用户手册

## 一、开箱与首次使用

1. 取出小爱音箱主机、电源适配器和说明书。
2. 将音箱放在桌面、床头柜或客厅中间位置，避免被柜体遮挡麦克风。
3. 接通电源，等待指示灯或语音提示进入配网状态。
4. 手机安装米家 APP 或小爱音箱 App。
5. 按 App 提示添加设备，选择家庭 2.4GHz WiFi 或支持的网络。
6. 添加完成后设置设备名称和房间。

## 二、基础操作

### 2.1 语音唤醒

1. 对音箱说“小爱同学”。
2. 听到提示音后说出指令，例如“播放音乐”“明天早上七点叫我”。
3. 若唤醒不灵敏，靠近音箱或降低环境噪声。
4. 不想被唤醒时，可按麦克风静音键。

### 2.2 播放音乐和内容

1. 通过语音说出歌曲、歌手、专辑或电台名称。
2. 可说“暂停”“继续播放”“音量调到 30%”。
3. 在 App 中绑定音乐服务账号，可获得更完整内容。
4. 蓝牙播放时，手机需要先与音箱蓝牙配对。

### 2.3 控制智能设备

1. 在米家 APP 中先添加灯、插座、净化器、扫地机等设备。
2. 给设备设置清晰名称和房间，例如“卧室台灯”。
3. 对音箱说“小爱同学，打开卧室台灯”。
4. 若控制错设备，检查设备名称是否重复。

### 2.4 红外遥控

1. 仅支持红外的音箱型号可添加红外设备。
2. 在 App 中选择添加遥控器。
3. 选择电视、空调、机顶盒等设备类型和品牌。
4. 按页面测试开关、音量、温度等按键。

## 三、进阶设置

### 3.1 家庭成员和儿童使用

1. 在米家 APP 中添加家庭成员。
2. 按需共享音箱和智能设备控制权限。
3. 儿童内容、购物和支付类能力应谨慎开启。
4. 不再使用的成员权限应及时移除。

### 3.2 智能场景

1. 在米家 APP 中进入自动化。
2. 设置“如果门锁开门，则打开玄关灯”。
3. 设置“晚上 23:00 关闭客厅插座”。
4. 场景创建后先手动测试，再长期启用。

### 3.3 网络变更

1. 更换路由器或 WiFi 密码后，音箱可能离线。
2. 按说明让音箱重新进入配网状态。
3. 在 App 中重新选择 WiFi 并输入密码。
4. 配网问题可参考 [NET_GUIDE_002] 5G/2.4G WiFi 配网说明（避坑）。

## 四、模式说明

| 模式名 | 适用场景 | 操作方式 |
|---|---|---|
| 语音助手 | 问答、闹钟、天气、控制设备 | 说“小爱同学”唤醒 |
| 蓝牙音箱 | 手机本地音乐或视频外放 | 手机蓝牙连接音箱 |
| 红外遥控 | 控制传统电视、空调 | 支持机型在 App 中添加遥控器 |
| 麦克风静音 | 隐私或会议场景 | 按音箱麦克风静音键 |
| 米家场景 | 自动化控制 | 米家 APP 创建自动化 |

## 五、保养与维护

- 音箱应放在干燥通风位置，不要放在浴室、厨房水槽旁或窗台雨水处。
- 麦克风孔和扬声器开孔不要被布料、纸张或装饰物遮挡。
- 清洁时断开电源，用干布擦拭外壳。
- 电源适配器应使用原装或规格匹配产品。
- 长时间不用时，拔掉电源并防尘存放。

## 六、常见问题速查

| 问题 | 解决方法 |
|---|---|
| 叫小爱没反应 | 检查电源、麦克风静音、网络和环境噪声 |
| 不能控制灯 | 检查灯是否在米家 APP 在线，名称是否清晰 |
| 配网失败 | 使用 2.4GHz WiFi，关闭路由器隔离和弱信号环境 |
| 红外遥控无效 | 确认无遮挡，重新匹配遥控器码库 |
| 播放卡顿 | 检查 WiFi 信号、音乐服务和路由器带宽 |
