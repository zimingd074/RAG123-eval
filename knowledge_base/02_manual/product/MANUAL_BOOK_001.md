# RedmiBook 笔记本用户手册

## 一、开箱与首次使用

1. 取出笔记本、电源适配器、电源线和纸质资料。
2. 检查屏幕、键盘、触控板、转轴、接口和外壳是否完好。
3. 连接电源适配器，首次开机前建议充电 10 分钟以上。
4. 按电源键开机，按 Windows 引导设置地区、键盘布局、网络和账户。
5. 登录 Microsoft 账户或创建本地账户。
6. 进入系统后先完成 Windows 更新、驱动更新和安全软件设置。

## 二、基础操作

### 2.1 开关机与睡眠

1. 点击开始菜单 > 电源，可选择睡眠、关机或重启。
2. 合上屏幕通常进入睡眠，具体行为可在电源设置中调整。
3. 长按电源键强制关机只用于系统无响应场景，可能导致未保存文件丢失。

### 2.2 连接网络

1. 点击任务栏网络图标。
2. 选择家庭、办公或校园 WiFi 并输入密码。
3. 公共 WiFi 登录后，不建议进行网银、支付和敏感账号操作。
4. 网络异常时，先关闭再开启 WLAN，必要时重启路由器。

### 2.3 触控板和快捷键

1. 单指移动光标，单击选择。
2. 双指上下滑动页面，双指点击打开右键菜单。
3. 三指切换任务或显示桌面，具体手势可在设置中调整。
4. Fn 组合键可调节亮度、音量、键盘灯或投屏，具体以机型键帽标识为准。

### 2.4 外接显示器和投屏

1. 使用 HDMI、USB-C 或扩展坞连接显示器。
2. 按 Win + P 选择复制、扩展或仅第二屏幕。
3. 若无画面，检查线材、接口、显示器输入源和刷新率。
4. 长时间外接高分辨率显示器时，注意散热和电源连接。

## 三、进阶设置

### 3.1 电池和性能模式

1. 点击任务栏电池图标查看电量。
2. 外出办公选择省电或均衡模式。
3. 游戏、剪辑和编译时连接电源并选择高性能模式。
4. 电池长期插电使用时，关注系统或厂商工具中的电池保护设置。

### 3.2 系统更新和驱动

1. 进入设置 > Windows 更新。
2. 下载并安装安全更新和驱动更新。
3. 更新期间保持电量充足并连接电源。
4. 更新失败时先重启，再重新检查更新。

### 3.3 数据备份和恢复

1. 重要文档建议同步到云盘或外接硬盘。
2. 工作资料按项目文件夹管理，避免只放在桌面。
3. 出售、送修前备份数据并退出账号。
4. 系统异常时，优先使用 Windows 恢复功能，不要直接格式化硬盘。

## 四、模式说明

| 模式名 | 适用场景 | 操作方式 |
|---|---|---|
| 睡眠 | 临时离开、短时间暂停 | 合盖或开始菜单选择睡眠 |
| 关机 | 长时间不用或携带外出 | 开始菜单 > 电源 > 关机 |
| 省电模式 | 外出低电量 | 设置 > 系统 > 电源和电池 |
| 高性能模式 | 游戏、剪辑、编译 | 接入电源后在电源设置中选择 |
| 投影模式 | 外接显示器、会议演示 | Win + P |

## 五、保养与维护

- 机身散热口不要被被子、软垫或文件堵住。
- 键盘和屏幕使用软布清洁，不要把液体直接喷到键盘。
- 电源适配器发热属正常现象，但应避免被衣物覆盖。
- 长期不用时，将电量保持在 50% 左右并关机存放。
- 雷雨天气或电压不稳时，建议拔掉电源和外接线缆。

## 六、常见问题速查

| 问题 | 解决方法 |
|---|---|
| 开机黑屏 | 连接电源等待数分钟，长按电源键重启，检查屏幕亮度 |
| 风扇声音大 | 检查后台更新、病毒扫描、游戏和散热口遮挡 |
| 触控板失灵 | 检查快捷键是否关闭触控板，重启电脑或更新驱动 |
| WiFi 掉线 | 更新无线网卡驱动，重启路由器，切换网络频段 |
| 外接屏无信号 | 检查线材、接口、显示器输入源和 Win + P 设置 |
