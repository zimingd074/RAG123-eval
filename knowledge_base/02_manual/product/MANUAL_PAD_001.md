# 小米平板用户手册

## 一、开箱与首次使用

1. 取出平板主机、数据线、电源适配器或纸质资料，按包装清单核对配件。
2. 检查屏幕、机身边框、摄像头、USB Type-C 接口和扬声器开孔是否完好。
3. 长按电源键开机，按屏幕提示选择语言、地区、网络和隐私设置。
4. 连接稳定 WiFi 后登录小米账号，按需开启云同步、查找设备和应用备份。
5. 首次使用建议充电至 50% 以上，再进行系统更新和应用安装。
6. 若从旧平板或手机迁移数据，保持两台设备电量充足，并优先使用系统迁移工具。

## 二、基础操作

### 2.1 开关机与重启

1. 长按电源键，出现电源菜单后选择关机或重启。
2. 平板无响应时，长按电源键约 10 秒进行强制重启。
3. 强制重启不会主动清除数据，但未保存内容可能丢失。

### 2.2 连接 WiFi 和蓝牙

1. 进入设置 > WLAN，选择家庭或办公 WiFi 并输入密码。
2. 若路由器双频合一导致连接不稳，可先连接独立 2.4GHz 或 5GHz 网络。
3. 进入设置 > 蓝牙，打开蓝牙并选择键盘、鼠标、耳机或手写笔配件。
4. 配对失败时，先让配件进入配对模式，再在平板端重新搜索。

### 2.3 安装应用和管理桌面

1. 打开应用商店或可信应用来源安装应用。
2. 长按桌面图标可移动、卸载或创建文件夹。
3. 常用学习、办公、影音应用可放在底部 Dock 或第一页。
4. 不常用应用可关闭通知和后台自启动，减少耗电。

### 2.4 分屏和悬浮窗

1. 打开支持分屏的应用。
2. 从任务管理界面或侧边栏选择分屏。
3. 拖动中间分隔线调整两个应用窗口大小。
4. 支持悬浮窗的应用可拖出小窗，用于聊天、查资料或计算。

## 三、进阶设置

### 3.1 手写笔和键盘

1. 将手写笔吸附在平板对应位置或按说明完成蓝牙配对。
2. 进入设置检查手写笔电量、按键功能和压感状态。
3. 蓝牙键盘配对后，可在输入法设置中调整中英文切换和快捷键。
4. 若键盘保护套触点连接不稳，清洁触点后重新吸附。

### 3.2 儿童与学习场景

1. 为儿童使用创建独立账号或开启儿童空间。
2. 设置应用可用范围、使用时长和内容限制。
3. 网课期间开启勿扰模式，减少游戏和社交通知干扰。
4. 长时间学习时打开护眼模式，并保持合适观看距离。

### 3.3 文件管理和跨设备传输

1. 使用文件管理器按图片、文档、下载、压缩包分类查看文件。
2. 重要课件和照片建议同步到云端或外接存储。
3. 与小米手机互传时，可使用系统互传或云服务。
4. 出售、送修或转交他人前，应备份数据并退出小米账号。

## 四、模式说明

| 模式名 | 适用场景 | 操作方式 |
|---|---|---|
| 护眼模式 | 阅读、网课、夜间使用 | 设置 > 显示与亮度 > 护眼模式 |
| 深色模式 | 夜间浏览、低亮环境 | 设置 > 显示与亮度 > 深色模式 |
| 勿扰模式 | 网课、会议、考试 | 控制中心或设置 > 声音与振动 |
| 分屏模式 | 记笔记、查资料、写作业 | 任务管理或侧边栏进入分屏 |
| 省电模式 | 外出或低电量 | 设置 > 电池 > 省电模式 |

## 五、保养与维护

- 屏幕使用柔软干布擦拭，不要把清洁液直接喷到屏幕和接口。
- 平板长时间高亮播放视频或游戏会发热，应保持通风。
- 保护壳、贴膜、手写笔和键盘应选择对应机型尺寸。
- USB Type-C 接口有灰尘时，可用软毛刷轻扫，不要用金属物硬挑。
- 长期不用时，将电量保持在 50% 左右并关机存放。

## 六、常见问题速查

| 问题 | 解决方法 |
|---|---|
| 分屏找不到应用 | 该应用可能不支持分屏，尝试用悬浮窗或更新应用 |
| 手写笔断连 | 检查电量、吸附位置和蓝牙状态，必要时重新配对 |
| 键盘输入错乱 | 检查输入法、语言布局和键盘电量 |
| 平板发热 | 暂停游戏、视频导出、系统更新等高负载任务 |
| WiFi 不稳定 | 重启路由器和平板，区分 2.4GHz/5GHz 网络 |
