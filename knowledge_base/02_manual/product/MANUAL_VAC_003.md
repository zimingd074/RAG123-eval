# 石头扫地机保养维护手册

## 一、开箱与首次使用

石头 T7、T7S Plus、G10、G10S Pro 都需要定期维护主刷、边刷、尘盒、滤网、拖布和传感器。G10、G10S Pro 还需要维护基站清洁盘、清水箱、污水箱和基站滤网。基础使用流程见 [MANUAL_VAC_001] 石头扫地机用户手册（基础版）。

维护前应先暂停任务，让主机回到充电座或基站。清理主刷、边刷、尘盒和基站时，应避免水进入主机电机、充电触点和基站电气部件。

## 二、基础操作

### 2.1 清理尘盒

1. 打开主机上盖。
2. 取出尘盒。
3. 打开尘盒卡扣，将灰尘倒入垃圾桶。
4. 用干布擦拭尘盒外侧和进风口。
5. 将尘盒装回主机，确认卡扣到位。

T7、T7S Plus、G10 的尘盒需要人工定期倒尘。G10S Pro 支持自动集尘，但仍应定期检查主机尘盒是否堵塞。

### 2.2 清理滤网

1. 从尘盒上取下滤网。
2. 轻拍滤网，去除表面浮尘。
3. 若需要水洗，使用清水冲洗，不使用清洁剂。
4. 将滤网完全晾干后再装回。
5. 不使用潮湿滤网启动主机。

### 2.3 清理主刷

1. 翻转主机，打开主刷罩。
2. 取出主刷。
3. 剪开缠绕毛发，重点清理主刷两端轴承。
4. 检查刷体是否破损或变形。
5. 装回主刷并扣紧主刷罩。

T7 使用 DuPont 刷毛主刷，长毛缠绕后需要更频繁清理。T7S Plus、G10、G10S Pro 使用胶刷结构，毛发清理相对简单，但仍需检查两端轴承。

### 2.4 清理边刷

1. 取下或拧下边刷。
2. 清理边刷根部毛发。
3. 检查刷臂是否明显变形。
4. 将边刷装回并确认固定牢靠。

## 三、进阶设置

### 3.1 维护周期建议

| 部件 | 建议检查周期 | 建议处理 |
|---|---|---|
| 尘盒 | 每次或每 2-3 次清扫后 | 倒尘并擦拭进风口 |
| 滤网 | 每周检查 | 轻拍除尘，必要时水洗并完全晾干 |
| 主刷 | 每周检查 | 清理毛发，宠物家庭提高频率 |
| 边刷 | 每 2 周检查 | 清理缠绕，变形后更换 |
| 拖布 | 每次拖地后 | 清洗并晾干 |
| 传感器 | 每 2 周检查 | 用干布擦拭 |
| 充电触点 | 每月检查 | 用干布擦拭 |
| 基站污水箱 | 每次或每 2-3 次回洗后 | 倒掉污水并冲洗 |
| 基站清洁盘 | 每 1-2 周检查 | 清理泥沙和毛发 |
| G10S Pro 集尘袋 | 按 APP 提醒 | 装满后更换 |

### 3.2 传感器清洁

1. 用干净软布擦拭悬崖传感器。
2. 擦拭前向避障传感器和机械碰撞区域。
3. 擦拭 LDS 激光模组外壳，不按压激光头。
4. 擦拭回充传感器和充电触点。

传感器脏污可能导致误判悬崖、频繁绕行、回充失败或地图异常。

### 3.3 G10 / G10S Pro 基站维护

1. 取出清水箱，补充清水。
2. 取出污水箱，倒掉污水并冲洗。
3. 拆下清洁盘，清理泥沙、毛发和水垢。
4. 检查基站滤网和水路入口。
5. G10S Pro 检查集尘袋是否装满或未装好。

污水箱长时间不清理会产生异味。清洁盘堵塞可能导致拖布回洗异常。

## 四、模式说明

| 模式名 | 适用场景 | 操作方式 |
|---|---|---|
| 日常维护 | 每周快速检查耗材 | 清理尘盒、滤网、主刷、边刷 |
| 深度维护 | 吸力明显下降或拖地变差 | 检查风道、水路、拖布和传感器 |
| 宠物家庭维护 | 毛发较多 | 提高主刷、边刷、尘盒清理频率 |
| 基站维护 | G10 / G10S Pro | 清理清洁盘、污水箱、集尘袋 |
| 长期存放 | 超过 1 个月不用 | 清空水箱、清洗拖布并充电后关机 |

## 五、保养与维护

- 不使用腐蚀性清洁剂清洗水箱、拖布和基站。
- 滤网必须完全晾干后再装回，潮湿滤网会影响吸力并可能损伤风机。
- 主刷和边刷变形、断裂或严重磨损后应更换。
- 拖布长期发硬、异味明显或清洗后仍脏，应更换。
- G10S Pro 集尘袋装满后应及时更换，避免影响自动集尘。
- 自动上下水套件若出现漏水或排水异常，应停止使用并检查安装位置。

## 六、常见问题速查

| 问题 | 解决方法 |
|---|---|
| 吸力变弱 | 清理尘盒、滤网、主刷和风道，确认滤网完全干燥 |
| 拖地有异味 | 清洗拖布、污水箱和清洁盘，并晾干拖布 |
| 主刷异响 | 拆下主刷，清理两端轴承和缠绕毛发 |
| 回充失败 | 擦拭充电触点和回充传感器，清理基站前方障碍 |
| 基站洗拖布变差 | 清理清洁盘、污水箱、滤网和水路入口 |
