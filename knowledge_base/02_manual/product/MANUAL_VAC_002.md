# 石头扫地机进阶设置（定时/分区/禁区）

## 一、开箱与首次使用

本手册适用于石头 T7、T7S Plus、G10、G10S Pro 的地图和清扫策略设置。首次使用、开机、配网和基础清扫见 [MANUAL_VAC_001] 石头扫地机用户手册（基础版）。

进阶设置前需满足以下条件：

1. 主机已成功添加到米家 APP 或 Roborock APP。
2. 已完成至少一次完整建图。
3. 地图中房间轮廓清晰，没有明显漏扫区域。
4. APP 已开启地图保存或多地图管理功能。

## 二、基础操作

### 2.1 房间分割

1. 打开设备首页，进入地图管理或地图编辑。
2. 选择房间分割。
3. 在需要分开的区域之间画分割线，例如客厅和餐厅之间、卧室和衣帽间之间。
4. 保存后返回地图，确认房间是否分割成功。
5. 若分割失败，调整分割线位置，避开门洞、斜墙和家具边缘。

### 2.2 房间合并

1. 进入地图编辑。
2. 选择房间合并。
3. 勾选需要合并的相邻房间。
4. 保存后重新命名房间。
5. 若两个区域中间有墙体或门长期关闭，不建议强行合并。

### 2.3 房间命名

1. 在地图中点击目标房间。
2. 选择重命名。
3. 使用客厅、主卧、次卧、厨房、卫生间、玄关等稳定名称。
4. 保存后，后续可通过语音或 APP 快速选择目标房间。

## 三、进阶设置

### 3.1 定时清扫

1. 进入设备首页，选择定时或预约清扫。
2. 设置执行时间和重复周期。
3. 选择清扫区域：全屋、指定房间或指定分区。
4. 设置吸力、水量和清扫次数。
5. 保存后确认定时任务处于启用状态。

建议：

- 工作日白天可设置全屋清扫。
- 夜间任务建议使用安静或标准吸力。
- 厨房、餐厅、玄关可设置更高频率。
- G10 / G10S Pro 可结合拖布回洗频率，避免大面积湿拖时拖布过脏。

### 3.2 禁区设置

1. 进入地图编辑，选择禁区。
2. 在地图中框选不希望机器人进入的区域。
3. 保存后发起局部清扫，观察机器人是否绕行。

适合设置禁区的位置：

- 宠物食盆和饮水区
- 地面线缆密集区
- 矮柜底部容易卡住的位置
- 儿童玩具区
- 高门槛或易跌落区域

### 3.3 虚拟墙设置

1. 进入地图编辑，选择虚拟墙。
2. 在门口、走廊边界或开放式区域画线。
3. 保存后机器人会将该线视为不可跨越边界。

虚拟墙适合开放空间的临时隔断。若区域形状较复杂，应使用禁区而不是虚拟墙。

### 3.4 禁拖区设置

1. 进入地图编辑，选择禁拖区。
2. 框选地毯、宠物垫、儿童爬行垫等不适合拖地的位置。
3. 保存后选择扫拖任务，确认机器人在该区域只吸尘或绕开拖地。

T7 不支持拖布自动抬升，地毯区域更依赖禁拖区。T7S Plus、G10、G10S Pro 支持拖布抬升，但长毛地毯仍建议设置禁拖区。

### 3.5 多地图管理

1. 将机器人搬到另一楼层。
2. 在 APP 中选择新建地图或多地图管理。
3. 完成该楼层完整建图。
4. 为地图命名，例如一楼、二楼、地下室。
5. 每次搬运后确认 APP 是否切换到正确地图。

扫地机器人不能自动上下楼，多地图只解决地图识别和房间管理问题。

## 四、模式说明

| 模式名 | 适用场景 | 操作方式 |
|---|---|---|
| 房间清扫 | 指定卧室、客厅、厨房 | 地图中勾选房间并启动 |
| 划区清扫 | 餐桌下方、玄关、沙发前 | 在地图中框选区域 |
| 定时清扫 | 每天固定时间清洁 | 设置预约任务和重复周期 |
| 禁区 | 完全不进入某区域 | 地图编辑中框选禁区 |
| 禁拖区 | 地毯等只吸不拖区域 | 地图编辑中框选禁拖区 |
| 多地图 | 多楼层住宅 | 在地图管理中保存多张地图 |

## 五、保养与维护

进阶设置本身不需要耗材维护，但地图准确性会受环境变化影响。若家具大幅移动、门长期关闭、基站位置改变，应重新建图或局部修正地图。基站位置改变后，应让主机从基站出发重新完成一次清扫，避免回充失败。

## 六、常见问题速查

| 问题 | 解决方法 |
|---|---|
| 房间无法分割 | 调整分割线位置，避开门洞和家具边缘 |
| 禁区不生效 | 确认已保存地图，并从同一张地图发起任务 |
| 定时任务没执行 | 检查主机电量、网络状态和定时任务开关 |
| 地毯被拖湿 | 为地毯区域设置禁拖区，长毛地毯不依赖自动抬升 |
| 多地图错乱 | 搬到对应楼层后手动切换地图，再启动清扫 |
