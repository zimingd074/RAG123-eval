# 小米智能门锁用户手册

## 一、开箱与首次使用

小米智能门锁属于入户安全设备，安装、初始化和权限管理应由房屋所有人或被授权家庭成员完成。首次使用前，应由安装人员完成锁体安装、开门方向配置、门缝调试和基础功能测试。

首次使用检查：

1. 检查门锁能否正常关门、上锁和开门。
2. 检查机械钥匙是否可用，并妥善放在门外可取的安全位置。
3. 检查电池是否安装到位。
4. 检查米家 APP 是否已绑定到本人账号。
5. 添加至少两种开锁方式，例如指纹和密码，避免单一方式失效。

## 二、基础操作

### 2.1 添加到米家 APP

1. 打开米家 APP。
2. 点击添加设备，搜索对应门锁型号。
3. 按 APP 提示靠近门锁，完成蓝牙连接。
4. 绑定到本人小米账号。
5. 设置设备名称和所在房间。

门锁类设备不建议绑定到临时账号或非家庭成员账号。

### 2.2 添加指纹

1. 进入门锁设备页。
2. 选择用户管理。
3. 选择目标家庭成员。
4. 点击添加指纹。
5. 按 APP 和门锁语音提示多次按压指纹识别区。
6. 保存后进行一次开锁测试。

### 2.3 添加密码

1. 进入用户管理。
2. 选择目标用户。
3. 添加长期密码或周期密码。
4. 避免使用生日、手机号后六位、连续数字等弱密码。
5. 保存后测试开锁。

### 2.4 使用机械钥匙

1. 找到门锁应急钥匙孔。
2. 插入机械钥匙并按正确方向转动。
3. 开门后检查电池和门锁状态。

机械钥匙用于本人应急开锁，不作为日常开锁方式。

## 三、进阶设置

### 3.1 用户权限管理

1. 在米家 APP 中进入用户管理。
2. 为家庭成员建立独立用户。
3. 给每个用户分别添加指纹、密码或 NFC。
4. 不再使用的用户权限应及时删除。
5. 租客、保洁、访客应优先使用临时密码或周期密码。

### 3.2 门锁通知

1. 在设备设置中开启开锁通知、低电量提醒、防撬提醒。
2. 检查手机通知权限。
3. 若需要远程通知，确认门锁已连接可用网关或具备远程能力。
4. 定期查看异常记录。

### 3.3 场景联动

1. 在米家 APP 中创建自动化。
2. 可设置“回家开锁后打开玄关灯”。
3. 可设置“离家上锁后关闭灯光和插座”。
4. 涉及安防的自动化应谨慎测试，避免误触发。

## 四、模式说明

| 模式名 | 适用场景 | 操作方式 |
|---|---|---|
| 指纹开锁 | 家庭成员日常进出 | 录入指纹后按压识别区 |
| 密码开锁 | 家庭成员或固定人员 | 输入已设置密码 |
| 临时密码 | 访客、保洁、亲友临时进门 | 米家 APP 创建并限定有效期 |
| NFC 开锁 | 支持的小米穿戴或卡片 | 在 APP 中添加对应设备 |
| 机械钥匙 | 电量耗尽或电子部分异常 | 使用应急钥匙 |
| 米家联动 | 回家/离家自动化 | 米家 APP 创建场景 |

## 五、保养与维护

- 定期检查电池电量，低电量提醒出现后及时更换或充电。
- 指纹识别区、密码区应保持干燥和清洁。
- 门体变形、门缝变化会影响锁舌伸缩，应及时调整门扣板。
- 机械钥匙应分开放置，不应全部放在屋内。
- 不使用腐蚀性清洁剂擦拭面板。

## 六、常见问题速查

| 问题 | 解决方法 |
|---|---|
| 指纹识别失败 | 擦干手指和识别区，重新录入常用手指 |
| 临时密码不能用 | 检查有效期、时区、网络同步和门锁时间 |
| APP 看不到门锁状态 | 靠近蓝牙连接，或检查网关和网络 |
| 门没关紧却显示异常 | 检查门缝、锁舌和门扣板位置 |
| 低电量后还能开门吗 | 优先更换电池，必要时使用 Type-C 应急供电或机械钥匙 |
