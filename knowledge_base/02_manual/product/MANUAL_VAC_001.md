# 石头扫地机用户手册（基础版）

## 一、开箱与首次使用

1. 取出主机、充电座或基站、电源线、拖布组件、说明书等配件。
2. 将充电座或基站靠墙放置，左右预留空间，前方保持无遮挡。
3. 接通电源后，把主机放回充电座或基站，确认充电触点接触正常。
4. 首次使用前建议充满电。T7 充满约需 6 小时，T7S Plus 约需 3 小时，G10 / G10S Pro 以基站回充显示为准。
5. 安装拖布前先确认水箱和拖布支架固定到位。带自清洁基站的 G10 / G10S Pro 需要向清水箱加入清水。

## 二、基础操作

### 2.1 开机与关机

1. 长按主机电源键，指示灯亮起后完成开机。
2. 主机处于非充电状态时，长按电源键可关机。
3. 主机正在充电、回充或执行任务时，不建议强制关机。

### 2.2 添加到 APP

1. 打开米家 APP 或 Roborock APP。
2. 点击添加设备，选择对应型号。
3. 按 APP 提示重置 WiFi，等待主机进入配网状态。
4. 选择 2.4GHz WiFi 并输入密码。
5. 添加完成后，为设备命名并选择所在房间。

配网失败时，优先确认路由器是否开启 2.4GHz WiFi。详细处理见 [NET_GUIDE_002] 5G/2.4G WiFi 配网说明。

### 2.3 首次建图

1. 清理地面上的数据线、轻薄地毯、宠物玩具和易倒物品。
2. 打开需要清扫的房门，关闭不希望进入的区域。
3. 在 APP 中选择全屋清扫或快速建图。
4. 建图完成后检查房间分区，必要时进行合并、分割或重命名。
5. 设置禁区、虚拟墙、禁拖区和地毯策略。

### 2.4 开始清扫

1. 在 APP 首页选择设备。
2. 选择全屋清扫、房间清扫、划区清扫或定点清扫。
3. 根据地面情况选择吸力档位和水量档位。
4. 点击开始，主机会离开充电座或基站执行任务。
5. 清扫完成后，主机会自动回充；G10 / G10S Pro 会按设置回洗拖布。

## 三、进阶设置

### 3.1 分区与房间管理

1. 打开地图管理。
2. 选择房间分割或房间合并。
3. 为厨房、客厅、卧室、玄关等区域命名。
4. 为不同房间设置独立吸力、水量和清扫次数。

### 3.2 禁区与禁拖区

1. 进入地图编辑。
2. 对地毯、宠物食盆区、线缆集中区设置禁区或禁拖区。
3. 保存地图后重新发起清扫，确认机器人是否绕行。
4. 地毯区域若需要吸尘但不拖地，应设置禁拖区而不是完全禁区。

### 3.3 定时清扫

1. 在 APP 中进入定时任务。
2. 选择执行日期、时间、清扫区域和清扫模式。
3. 对夜间任务选择低噪音吸力档位。
4. 保存后确认定时任务是否处于启用状态。

### 3.4 基站设置

1. G10 可设置拖布回洗频率和补水策略。
2. G10S Pro 可设置自动集尘频率、热风烘干时长、拖布回洗频率。
3. 使用自动上下水套件时，应在 APP 中确认上下水模块状态。
4. 基站工作异常时，先检查水箱、集尘袋、清洁盘和电源连接。

## 四、模式说明

| 模式名 | 适用场景 | 操作方式 |
|---|---|---|
| 全屋清扫 | 日常完整清扫 | APP 首页选择全屋清扫 |
| 房间清扫 | 指定卧室、客厅、厨房 | 地图中勾选目标房间 |
| 划区清扫 | 局部脏污、餐桌下方 | 在地图上框选区域 |
| 定点清扫 | 小范围碎屑 | 将主机移动到附近后启动定点清扫 |
| 禁拖清扫 | 地毯、宠物垫区域 | 在地图中设置禁拖区 |
| 回充 | 清扫结束或电量不足 | APP 点击回充或主机自动回充 |

## 五、保养与维护

- 主刷：每周检查一次缠绕毛发，宠物家庭可提高到每 2-3 天一次。
- 边刷：每 2 周检查一次变形和缠绕情况。
- 滤网：可水洗滤网需完全晾干后再装回，建议准备备用滤网轮换。
- 拖布：T7 / T7S Plus 使用后应取下清洗并晾干；G10 / G10S Pro 也应定期取下深度清洗。
- 尘盒：T7、T7S Plus、G10 需要人工倒尘；G10S Pro 使用集尘袋，但仍需检查主机尘盒是否堵塞。
- 基站：G10 / G10S Pro 需要定期清理清洁盘、污水箱和基站滤网。

## 六、常见问题速查

| 问题 | 解决方法 |
|---|---|
| 找不到设备 | 确认手机和设备连接 2.4GHz WiFi，并按说明重置设备 WiFi |
| 建图不完整 | 首次建图前打开房门并移除地面障碍，必要时重新建图 |
| 回充失败 | 检查充电座或基站周围是否有障碍物，确认充电触点干净 |
| 拖地水量小 | 检查水箱是否有水、拖布是否安装到位、水路是否堵塞 |
| 吸力变弱 | 清理尘盒、滤网、主刷和风道，确认未使用潮湿滤网 |
