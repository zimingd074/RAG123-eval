# 小米智能门锁进阶（指纹/临时密码）

## 一、开箱与首次使用

本手册适用于小米智能门锁 E10 和小米全自动智能门锁 Pro 的用户权限管理。基础安装、绑定和日常开锁见 [MANUAL_LOCK_001] 小米智能门锁用户手册。

进阶设置前确认：

1. 门锁已绑定到本人米家账号。
2. 管理员身份可正常进入用户管理页面。
3. 门锁电量充足。
4. 手机与门锁保持近距离蓝牙连接，或远程能力可用。

## 二、基础操作

### 2.1 指纹录入策略

1. 每名家庭成员至少录入 2 个常用手指。
2. 老人或指纹较浅用户，可多录入同一手指的不同按压角度。
3. 录入时保持手指干燥，不要只录入指尖边缘。
4. 保存后立即测试 3 次，确认识别稳定。

### 2.2 删除指纹

1. 打开米家 APP，进入门锁设备页。
2. 进入用户管理。
3. 选择目标用户。
4. 删除不再使用的指纹。
5. 保存后查看权限列表。

离职保洁、退租租客、离开家庭组成员的权限应及时删除。

### 2.3 设置临时密码

1. 进入门锁设备页。
2. 选择临时密码或密码管理。
3. 选择一次性密码或周期密码。
4. 设置有效时间。
5. 将密码通过可信渠道发给访客。
6. 访客使用后，在 APP 中检查开锁记录。

### 2.4 设置周期密码

1. 选择周期密码。
2. 设置开始日期、结束日期和可用时间段。
3. 适用于保洁、钟点工、固定上门服务等场景。
4. 服务结束后删除周期密码。

## 三、进阶设置

### 3.1 虚位密码使用

虚位密码允许在真实密码前后输入额外数字，降低旁人窥视风险。例如真实密码为 123456，可输入 88812345699，只要连续包含真实密码即可识别。

使用建议：

- 不要长期使用过短密码。
- 不要使用生日、门牌号、手机号后几位。
- 在有人旁观时使用虚位密码。

### 3.2 NFC 设备管理

1. 在用户管理中选择添加 NFC 设备。
2. 按 APP 提示将小米手环、小米手表或 NFC 卡片靠近识别区。
3. 添加成功后进行一次开锁测试。
4. NFC 设备丢失后，应立即在门锁中删除对应权限。

### 3.3 访客权限边界

1. 访客优先使用一次性密码。
2. 长期服务人员使用周期密码，不使用管理员权限。
3. 不把家庭成员账号借给外部人员。
4. 不在公开聊天群发送门锁密码。

### 3.4 异常记录检查

1. 定期查看开锁记录。
2. 关注连续验证失败、防撬报警、门虚掩等记录。
3. 若出现未知用户开锁记录，应立即修改密码并检查权限列表。
4. 必要时联系售后或物业协助排查。

## 四、模式说明

| 模式名 | 适用场景 | 操作方式 |
|---|---|---|
| 多指纹录入 | 家庭成员日常使用 | 为同一用户录入多个手指 |
| 一次性密码 | 临时访客 | 设置一次有效密码 |
| 周期密码 | 保洁、钟点工 | 设置日期和时间段 |
| 虚位密码 | 防窥输入 | 在真实密码前后加入随机数字 |
| NFC 权限 | 手环/手表开锁 | 添加支持的 NFC 设备 |
| 权限清理 | 人员变动 | 删除不再使用的用户和凭证 |

## 五、保养与维护

- 定期检查用户列表，清理不再使用的权限。
- 指纹识别区若有水渍、油污、灰尘，应使用干软布擦拭。
- 临时密码使用后，检查是否仍有未删除的周期权限。
- NFC 设备丢失后，应第一时间删除权限。
- 管理员手机更换或账号迁移前，应确认仍有可用管理员权限。

## 六、常见问题速查

| 问题 | 解决方法 |
|---|---|
| 老人指纹不好识别 | 多录入同一手指不同角度，并保留密码或 NFC 方式 |
| 临时密码过期 | 重新创建有效期内的临时密码 |
| 保洁每周固定来 | 使用周期密码并限定日期和时间段 |
| NFC 设备丢失 | 立即删除对应 NFC 权限 |
| 开锁记录有陌生记录 | 立即修改密码、清理权限并检查家庭账号成员 |
