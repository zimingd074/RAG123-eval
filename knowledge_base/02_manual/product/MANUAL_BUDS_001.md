# 小米/Redmi Buds 用户手册（含触控操作）

## 一、开箱与首次使用

1. 取出左右耳机、充电盒、耳帽或充电线和说明书。
2. 检查耳机触点、充电盒接口和耳帽是否完好。
3. 首次使用前，将耳机放入充电盒并充电 10 分钟以上。
4. 打开充电盒上盖，让耳机进入配对状态。
5. 手机打开蓝牙，选择对应耳机名称完成连接。
6. 小米手机可按弹窗提示快速连接，非小米设备在蓝牙列表中手动配对。

## 二、基础操作

### 2.1 蓝牙配对

1. 打开充电盒上盖。
2. 长按充电盒按键，直到指示灯闪烁或耳机进入配对状态。
3. 手机进入设置 > 蓝牙。
4. 选择耳机名称并确认连接。
5. 配对成功后，耳机会自动记忆常用设备。

### 2.2 佩戴和耳帽选择

1. 入耳式耳机应选择合适尺寸耳帽。
2. 将耳机轻轻旋入耳道，保证密封但不胀痛。
3. 半入耳耳机无需耳帽，佩戴时调整柄部方向。
4. 运动前轻晃头部确认耳机不易掉落。

### 2.3 触控或按压操作

1. 双击通常用于播放/暂停或接听电话。
2. 长按通常用于切换降噪、通透或拒接电话。
3. 三击或捏柄操作可按机型设置为下一曲、上一曲或语音助手。
4. 具体手势以耳机型号和米家/小米耳机设置页为准。

### 2.4 充电

1. 将耳机放回充电盒，确认触点吸附到位。
2. 使用 USB Type-C 线为充电盒充电。
3. 支持无线充电的型号可放在 Qi 充电板上。
4. 耳机或充电盒潮湿时，不要立即充电。

## 三、进阶设置

### 3.1 降噪和通透模式

1. 在耳机设置页选择降噪、通透或关闭。
2. 地铁、飞机、办公室可开启降噪。
3. 户外跑步、骑行或过马路时建议使用通透模式或降低音量。
4. 半入耳降噪效果受佩戴贴合影响更明显。

### 3.2 双设备连接

1. 仅支持该功能的型号可开启双设备连接。
2. 先分别与手机和电脑配对。
3. 在耳机设置中开启双设备连接。
4. 来电、会议和音乐切换时，按设备优先级自动切换。

### 3.3 查找耳机

1. 打开耳机设备页。
2. 选择查找耳机。
3. 让耳机播放提示音。
4. 不要在佩戴状态下播放高音量查找提示。

## 四、模式说明

| 模式名 | 适用场景 | 操作方式 |
|---|---|---|
| 普通模式 | 安静环境听歌、通话 | 默认模式 |
| 主动降噪 | 通勤、飞机、办公室 | 长按耳机或 App 中切换 |
| 通透模式 | 户外行走、需要听环境音 | 长按耳机或 App 中切换 |
| 低延迟模式 | 游戏、视频 | 支持机型在 App 或系统中开启 |
| 双设备连接 | 手机和电脑同时使用 | 支持机型在设置中开启 |

## 五、保养与维护

- 耳帽、出音孔和充电触点应定期清洁。
- 运动后擦干汗水，再放入充电盒。
- IPX4/IP54 只防日常汗水和轻微泼溅，不可游泳或淋浴。
- 充电盒不参与多数防水等级，不要进水。
- 长期不用时，每 1-2 个月给耳机和充电盒补电一次。

## 六、常见问题速查

| 问题 | 解决方法 |
|---|---|
| 只有一边有声音 | 将双耳放回盒内重置，再重新配对 |
| 降噪效果差 | 更换耳帽尺寸，确认佩戴密封，检查是否开启降噪 |
| 触控不灵 | 擦干触控区和手指，检查是否误触禁用 |
| 充不进电 | 清洁耳机和充电盒触点，换线材和充电器测试 |
| 连接断续 | 远离强干扰环境，删除蓝牙记录后重新配对 |
