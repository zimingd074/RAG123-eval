# 米家电动牙刷使用手册

## 一、开箱与首次使用

1. 取出牙刷手柄、刷头、保护盖、识别环、充电底座和说明书。
2. 检查刷头、手柄传动轴和充电底座是否完好。
3. 将刷头垂直插入传动轴，刷头与手柄之间保留少量间隙属正常结构。
4. 首次使用前可将手柄放在充电底座上补电。
5. 下载米家 APP，按提示添加牙刷设备。
6. 初次使用建议选择轻柔模式，逐步适应震动。

## 二、基础操作

### 2.1 开始刷牙

1. 湿润刷头并挤上适量牙膏。
2. 将刷头放入口中后再按电源键。
3. 让刷毛轻贴牙面，不要像手动牙刷一样用力横向刷。
4. 按牙齿外侧、内侧、咬合面分区移动。
5. 刷牙结束后关机，再从口中取出牙刷。

### 2.2 切换模式

1. 短按按键开机。
2. 按说明书在标准、轻柔和自定义模式之间切换。
3. 牙龈敏感或初次使用时选择轻柔模式。
4. 自定义模式可在米家 APP 中设置强度和时长。

### 2.3 查看刷牙数据

1. 打开米家 APP。
2. 进入牙刷设备页。
3. 查看刷牙时长、覆盖率、过压提醒和历史记录。
4. 数据仅用于习惯参考，不作为牙科诊断依据。

### 2.4 充电

1. 将充电底座放在干燥平整位置。
2. 将牙刷手柄垂直放入充电底座。
3. 观察指示灯确认充电状态。
4. 充电底座和插头不要放在积水区域。

## 三、进阶设置

### 3.1 过压提醒

1. 刷牙时用力过大，牙刷会通过灯光或震动节奏提醒。
2. 出现过压提醒时，减轻手部压力。
3. 长期过压会刺激牙龈，不建议追求“刷得越用力越干净”。

### 3.2 刷头识别和家庭使用

1. 家庭多人使用时，为每个人分配独立刷头。
2. 使用彩色识别环区分刷头。
3. 刷头不可共用，使用后应冲洗并晾干。
4. 儿童使用需由成人指导。

### 3.3 刷头更换

1. 刷毛外翻、变形、变色或磨损时应更换。
2. 常规建议约 3 个月更换一次刷头。
3. 感冒、口腔炎症或刷头污染后可提前更换。
4. 更换时选择米家 T500 官方适配刷头。

## 四、模式说明

| 模式名 | 适用场景 | 操作方式 |
|---|---|---|
| 标准模式 | 日常清洁 | 按键切换至标准模式 |
| 轻柔模式 | 初次使用、牙龈敏感 | 按键切换至轻柔模式 |
| 自定义模式 | 个人偏好强度和时长 | 米家 APP 设置 |
| 过压提醒 | 刷牙用力过大 | 自动触发提醒 |
| 刷牙记录 | 查看习惯和覆盖情况 | 米家 APP 查看 |

## 五、保养与维护

- 每次使用后冲洗刷头和手柄连接处，并甩干水分。
- 刷头保护盖用于短途收纳，日常存放应保持通风。
- 充电底座保持干燥，避免牙膏泡沫和积水长期残留。
- 不要用热水煮刷头或将手柄放入消毒柜。
- 长期不用时，清洁晾干后存放，并定期补电。

## 六、常见问题速查

| 问题 | 解决方法 |
|---|---|
| 初次使用牙龈不适 | 改用轻柔模式，减少用力，持续不适应咨询牙医 |
| 刷牙数据不同步 | 打开米家 APP，靠近牙刷，检查蓝牙和账号 |
| 充不进电 | 擦干手柄底部和充电底座，重新放置 |
| 刷头松动 | 确认刷头型号和安装方向，少量间隙属正常 |
| 震动变弱 | 检查电量、模式和刷头是否老化 |
