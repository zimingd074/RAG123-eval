# 小米手机用户手册（通用）

## 一、开箱与首次使用

1. 取出手机、电源适配器、USB Type-C 数据线、保护壳、取卡针和纸质资料。
2. 检查机身、屏幕、摄像头玻璃和充电口是否完好。
3. 使用取卡针打开卡托，按卡托标识放入 Nano-SIM 卡；混合卡槽机型按商品规格选择 SIM 卡或存储卡。
4. 长按电源键开机，按屏幕提示选择语言、地区、网络和隐私设置。
5. 登录小米账号，按需开启云服务、查找设备、短信同步和照片同步。
6. 从旧手机迁移数据时，优先使用系统迁移工具或云备份；迁移过程中保持两台设备电量充足。
7. 首次使用前建议充电至 50% 以上，再进行系统更新、应用安装和数据迁移。

## 二、基础操作

### 2.1 开关机与重启

1. 长按电源键，屏幕出现电源菜单后选择关机或重启。
2. 手机无响应时，同时长按电源键约 10 秒触发强制重启。
3. 强制重启不会主动清除用户数据，但未保存的编辑内容可能丢失。

### 2.2 连接移动网络和 WiFi

1. 进入设置，打开移动网络，确认 SIM 卡已识别。
2. 根据运营商套餐开启 5G、VoLTE 或数据漫游。
3. 进入 WLAN，选择家庭或办公网络并输入密码。
4. 若使用 WiFi 6、WiFi 7 或 5GHz 网络，需要路由器和手机同时支持。
5. 公共 WiFi 登录后不要进行支付、网银等敏感操作。

### 2.3 录入指纹和人脸

1. 进入设置 > 密码与安全。
2. 先设置锁屏密码、图案或数字密码。
3. 按提示录入指纹；屏下指纹机型需覆盖完整识别区域，侧边指纹机型需多角度触碰按键。
4. 按提示录入人脸，避免强逆光、口罩遮挡或过暗环境。
5. 若识别率下降，删除旧数据后重新录入。

### 2.4 拍照和录像

1. 打开相机，选择拍照、人像、夜景、录像或专业模式。
2. 使用 0.6x、1x、2x、3.2x 等焦段时，以机型实际镜头配置为准。
3. 拍摄运动对象时，优先使用高快门、连拍或运动追焦功能。
4. 录制 4K 或 8K 视频前，检查剩余存储空间和电量。
5. 长时间录像会导致发热，必要时降低分辨率或暂停散热。

### 2.5 充电与电量管理

1. 使用随机附带或符合规格的电源适配器和数据线。
2. 插入 USB Type-C 线时不要大力晃动接口。
3. 开启快充时保持手机背面和充电器通风。
4. 手机明显发热时，暂停游戏、导航、热点和录像等高负载任务。
5. 无线充电机型应让手机线圈与无线充电器中心对齐，保护壳过厚时可先取下。

## 三、进阶设置

### 3.1 系统更新

1. 进入设置 > 我的设备 > 系统版本。
2. 连接稳定 WiFi，并保持电量高于 30%。
3. 下载更新包后按提示安装。
4. 更新期间不要强制关机、拔卡或清理系统文件。
5. 更新后如出现短暂发热或耗电增加，可等待系统完成后台优化。

### 3.2 应用权限管理

1. 进入设置 > 应用设置 > 权限管理。
2. 对相机、麦克风、位置、通讯录和通知权限逐项检查。
3. 对不常用应用关闭自启动、后台定位和后台联网。
4. 支付、门禁、车钥匙等 NFC 功能需要单独按应用提示授权。

### 3.3 显示与刷新率

1. 进入设置 > 显示与亮度。
2. 按需选择自动亮度、护眼模式、深色模式和色彩风格。
3. 高刷新率可提升流畅度，但会增加耗电。
4. 2K 屏机型可在清晰度和续航之间按场景切换。

### 3.4 数据备份与查找设备

1. 登录小米账号。
2. 开启查找设备、云备份、相册同步和联系人同步。
3. 更换手机前，先在旧手机完成备份，再在新手机恢复。
4. 出售或送修前，备份数据后退出小米账号并清除锁屏密码。

## 四、模式说明

| 模式名 | 适用场景 | 操作方式 |
|---|---|---|
| 省电模式 | 电量不足、外出无法及时充电 | 设置 > 电池 > 省电模式 |
| 超级省电 | 极低电量，仅保留基础通信 | 设置 > 电池 > 超级省电 |
| 勿扰模式 | 会议、睡眠、学习 | 控制中心或设置 > 声音与触感 |
| 护眼模式 | 夜间阅读、长时间看屏幕 | 设置 > 显示与亮度 > 护眼模式 |
| 专注模式 | 减少通知打扰 | 设置 > 屏幕时间管理或相关系统工具 |
| 性能模式 | 游戏、剪辑、跑分等高负载 | 设置 > 电池 > 性能模式，部分机型支持 |

## 五、保养与维护

- 机身和屏幕使用柔软干布清洁，不使用酒精大量直接喷洒屏幕、听筒和充电口。
- 手机进液、受潮或充电口有水迹时，停止充电并自然晾干，必要时联系售后。
- 电池适合在常温环境下使用，避免长时间放在车内暴晒、暖气旁或低温户外。
- 长期不用时，将电量保持在 50% 左右并关机存放，每 2-3 个月补电一次。
- 保护壳、贴膜、充电器、数据线应选择对应机型和正规规格。
- 充电异常、发热异常或无法开机时，参考 [CODE_PHONE_001] 小米手机充电异常排查（含发热警告）。

## 六、常见问题速查

| 问题 | 解决方法 |
|---|---|
| 新机首次开机发热 | 数据迁移、系统更新和应用恢复会带来高负载，完成后观察 1-2 小时 |
| 指纹识别不灵敏 | 清洁手指和识别区域，重新录入指纹，检查贴膜是否过厚 |
| WiFi 能连但网速慢 | 重启路由器，切换 2.4GHz/5GHz，检查是否多人占用带宽 |
| 充电速度低于标称 | 检查充电器、线材、温度、电量区间和是否边充边用 |
| 拍照占用空间过大 | 关闭不必要的高像素、RAW、8K 视频，定期备份相册 |
