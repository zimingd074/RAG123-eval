# 小米手表用户手册（含表盘自定义）

> 本手册适用于小米手表系列（S3 等），各型号在基础操作、健康监测、表盘自定义和日常维护等功能上操作方式一致。

## 一、开箱与首次使用

1. 取出手表主机、表带、充电底座或充电线和说明书。
2. 检查屏幕、表冠、按键、传感器区域和充电触点是否完好。
3. 将手表放到原装充电底座上充电，首次使用建议充至 50% 以上。
4. 长按电源键开机，按屏幕提示选择语言。
5. 在手机上安装小米运动健康 / Mi Fitness 或对应提示的 App。
6. 打开 App，扫描手表二维码或搜索设备完成绑定。

## 二、基础操作

### 2.1 佩戴和解锁

1. 将表带佩戴在腕骨上方约一指位置。
2. 表带松紧以传感器贴合皮肤但不勒手为准。
3. 抬腕亮屏或按键亮屏。
4. 运动时可适当收紧表带，提高心率数据稳定性。

### 2.2 查看通知和接听电话

1. 在手机 App 中开启通知权限。
2. 选择需要同步通知的应用。
3. 手表收到通知后可在通知列表查看。
4. 支持蓝牙通话的机型需保持手机蓝牙连接。

### 2.3 记录运动

1. 在手表中打开运动应用。
2. 选择跑步、骑行、游泳、徒步或其他运动模式。
3. 户外运动等待定位成功后再开始。
4. 运动结束后保存记录，并在 App 中查看轨迹、心率和消耗数据。

### 2.4 充电

1. 使用原装充电底座或磁吸充电线。
2. 将手表背面触点与充电器对齐。
3. 充电前擦干手表和充电触点。
4. 不建议在高温、潮湿或阳光直射环境中充电。

## 三、进阶设置

### 3.1 自定义表盘

1. 打开手机端 App。
2. 进入设备页 > 表盘市场。
3. 选择喜欢的表盘并下载同步。
4. 部分表盘支持自定义组件，可调整步数、心率、天气、电量等显示项。
5. 长按手表当前表盘，也可在手表端切换已下载表盘。

### 3.2 健康监测

1. 在 App 中进入健康监测设置。
2. 按需开启全天心率、睡眠、血氧、压力等监测。
3. 全天监测会增加耗电，若续航压力大可降低频率或关闭部分项目。
4. 健康数据仅供日常参考，不作为诊断或治疗依据。

### 3.3 运动目标和提醒

1. 在 App 中设置每日步数、活动热量或站立提醒。
2. 久坐提醒适合办公人群。
3. 运动自动识别会增加后台耗电。
4. 训练计划应根据自身身体状态安排，不应只依赖手表提示。

## 四、模式说明

| 模式名 | 适用场景 | 操作方式 |
|---|---|---|
| 运动模式 | 跑步、骑行、游泳、力量训练 | 手表运动应用中选择 |
| 勿扰模式 | 睡眠、会议 | 控制中心开启 |
| 剧场模式 | 暗光环境避免亮屏 | 控制中心开启 |
| 省电模式 | 低电量或长时间外出 | 设置 > 电池 |
| 常亮显示 | 需要随时看时间 | 表盘设置中开启，耗电增加 |

## 五、保养与维护

- 运动后擦干表身和表带，尤其是传感器和充电触点。
- 皮革表带不建议游泳、淋雨或大量出汗时佩戴。
- 5 ATM 防水不代表可用于热水、桑拿、潜水或高速水流冲击。
- 表带老化、开裂或卡扣松动时应及时更换。
- 长期不用时，将电量保持在 50% 左右并关机存放。

## 六、常见问题速查

| 问题 | 解决方法 |
|---|---|
| 手表收不到通知 | 检查手机蓝牙、App 后台权限和通知权限 |
| 运动轨迹不准 | 户外运动等待定位成功，避开高楼密集区域 |
| 心率数据波动大 | 调整佩戴松紧，擦干手腕和传感器 |
| 续航变短 | 关闭常亮显示、全天血氧和高频心率监测 |
| 表盘同步失败 | 保持手机和手表靠近，检查网络和手表电量 |
