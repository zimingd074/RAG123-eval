# 小米手机操作进阶（截屏/分屏/快捷键）

## 一、开箱与首次使用

本手册适用于已完成基础开机、联网、账号登录和锁屏设置的小米 / Redmi 手机。首次使用进阶操作前，应先确认系统为 MIUI 14 或小米澎湃 OS，并完成以下准备：

1. 进入设置 > 我的设备，检查系统版本并安装可用更新。
2. 进入设置 > 桌面，确认系统导航方式是经典导航键还是全面屏手势。
3. 进入设置 > 更多设置 > 快捷手势，查看截屏、手电筒、相机等快捷动作是否开启。
4. 进入设置 > 通知与状态栏，确认控制中心和通知样式符合使用习惯。
5. 重要数据操作前，先参考 [MANUAL_PHONE_001] 小米手机用户手册（通用）完成数据备份。

## 二、基础操作

### 2.1 三指下滑截屏

1. 打开需要截屏的页面。
2. 使用三根手指从屏幕中上部向下滑动。
3. 屏幕出现缩略图后，点击可进入编辑、裁剪、涂鸦或分享。
4. 若三指下滑无效，进入设置 > 更多设置 > 快捷手势 > 截屏，开启三指下拉。

### 2.2 按键截屏

1. 同时按住电源键和音量减键。
2. 听到提示音或看到屏幕闪动后松开。
3. 在左下角缩略图中选择分享、编辑或滚动截屏。
4. 该方式适合游戏、视频页面或三指手势被应用拦截时使用。

### 2.3 长截屏

1. 先用三指下滑或按键方式完成普通截屏。
2. 在截屏缩略图出现时点击“截长屏”。
3. 等页面自动滚动到需要位置后点击结束。
4. 若页面不支持滚动，系统会只保存当前屏幕内容。

### 2.4 录屏

1. 从控制中心找到屏幕录制图标。
2. 点击开始录制，按提示授权麦克风或悬浮窗权限。
3. 录制时点击悬浮按钮可暂停或结束。
4. 录屏文件保存在相册或屏幕录制文件夹中。
5. 录制网课、会议或影视内容时，应遵守平台和隐私规则。

### 2.5 快速打开相机和手电筒

1. 进入设置 > 更多设置 > 快捷手势。
2. 设置双击电源键打开相机。
3. 设置双击音量减键或锁屏快捷入口打开手电筒，具体入口随系统版本变化。
4. 锁屏状态下使用快捷相机拍摄后，未解锁前通常只能查看本次拍摄内容。

## 三、进阶设置

### 3.1 分屏

1. 打开第一个需要使用的应用。
2. 进入最近任务界面。
3. 长按应用卡片，选择分屏图标。
4. 在下方选择第二个应用。
5. 拖动中间分隔条可调整两个应用的显示比例。

部分应用因安全、视频版权或界面适配原因不支持分屏。若分屏图标不可用，应换用小窗或普通切换。

### 3.2 小窗应用

1. 打开最近任务界面。
2. 选择小窗应用入口，或长按应用卡片选择小窗。
3. 拖动小窗顶部可移动位置。
4. 拖动边角可调整尺寸，点击全屏按钮可恢复全屏。
5. 聊天、计算器、便签、浏览器等轻任务更适合小窗。

### 3.3 控制中心快捷开关

1. 从屏幕右上方向下滑动打开控制中心。
2. 点击编辑或排序入口。
3. 添加常用开关，如手电筒、录屏、勿扰、热点、NFC、深色模式。
4. 长按部分开关可进入对应设置页面。
5. 将高频开关放在第一屏，减少多次滑动。

### 3.4 快捷手势和按键

1. 进入设置 > 更多设置 > 快捷手势。
2. 按需开启三指截屏、双击电源键打开相机、双击背部动作等功能。
3. 若误触频繁，可关闭低频快捷手势。
4. 使用保护壳后，检查电源键和音量键是否被挤压。

### 3.5 全面屏手势

1. 进入设置 > 桌面 > 系统导航方式。
2. 选择全面屏手势后，按教程学习返回、回到桌面和最近任务。
3. 返回手势通常从屏幕左右边缘向内滑动。
4. 回到桌面通常从底部向上滑动。
5. 最近任务通常从底部向上滑动并短暂停留。

## 四、模式说明

| 模式名 | 适用场景 | 操作方式 |
|---|---|---|
| 截屏模式 | 保存聊天记录、订单、网页、课程资料 | 三指下滑或电源键 + 音量减键 |
| 长截屏 | 保存长网页、长聊天记录、长文档 | 截屏缩略图 > 截长屏 |
| 录屏模式 | 记录操作流程、课程片段、故障现象 | 控制中心 > 屏幕录制 |
| 分屏模式 | 一边看资料一边记笔记 | 最近任务 > 分屏 |
| 小窗模式 | 聊天、计算器、临时网页查询 | 最近任务 > 小窗应用 |
| 勿扰模式 | 会议、课堂、睡眠 | 控制中心 > 勿扰 |
| 单手模式 | 大屏手机单手操作 | 设置 > 更多设置 > 单手模式，部分机型支持 |

## 五、保养与维护

- 快捷手势过多可能增加误触，建议只保留高频功能。
- 录屏和长截屏会占用存储空间，应定期清理不需要的文件。
- 分屏和小窗会增加系统负载，低内存版本同时运行大型应用时可能出现卡顿。
- 控制中心的 NFC、热点、蓝牙和定位不用时可关闭，减少耗电和误连接。
- 若系统更新后手势入口变化，应在设置中搜索“截屏”“小窗”“分屏”“快捷手势”。
- 操作中出现充电异常、过热或卡死，可分别参考 [CODE_PHONE_001] 小米手机充电异常排查（含发热警告）和 [MANUAL_PHONE_001] 小米手机用户手册（通用）。

## 六、常见问题速查

| 问题 | 解决方法 |
|---|---|
| 三指截屏没反应 | 进入设置 > 更多设置 > 快捷手势，确认三指下拉截屏已开启 |
| 长截屏按钮不出现 | 当前应用或页面不支持滚动截屏，可改用普通截屏分段保存 |
| 分屏图标是灰色 | 应用不支持分屏，改用小窗或普通切换 |
| 录屏没有声音 | 检查录屏设置中的声音来源和麦克风权限 |
| 全面屏手势误触 | 重新学习手势，或切换回经典导航键 |
