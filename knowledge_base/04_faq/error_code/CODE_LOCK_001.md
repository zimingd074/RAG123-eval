# 小米智能门锁错误码与异常提示

## 错误码总览

小米智能门锁在 APP 和门锁语音中更常见的是“异常提示”而非固定数字错误码。本手册按客服问答中高频提示归类，适用于小米智能门锁 E10 和小米全自动智能门锁 Pro。

| 错误码 | 含义 | 严重程度 | 自助可解 |
|---|---|---|---|
| LOCK_LOW_BATTERY | 门锁低电量 | 中 | 是 |
| VIDEO_LOW_BATTERY | 可视模块低电量 | 中 | 是 |
| VERIFY_FAILED | 指纹或密码验证失败 | 低 | 是 |
| LOCK_TEMP_LOCKED | 连续验证失败触发临时锁定 | 中 | 是 |
| DOOR_AJAR | 门未关好或门虚掩 | 中 | 是 |
| BOLT_BLOCKED | 锁舌受阻或上锁失败 | 高 | 部分可解 |
| PRY_ALARM | 防撬报警 | 高 | 部分可解 |
| NETWORK_OFFLINE | 门锁离线或状态不同步 | 中 | 是 |
| SENSOR_ERROR | 传感器异常 | 高 | 否 |
| MOTOR_ERROR | 全自动锁体电机异常 | 高 | 否 |

## LOCK_LOW_BATTERY：门锁低电量

### 触发条件

干电池电量不足，门锁通过语音、指示灯或米家 APP 发出低电量提醒。

### 排查步骤

1. 使用已授权方式开门。
2. 进入室内更换门锁干电池。
3. 检查电池正负极方向。
4. 更换后测试指纹、密码和上锁动作。

### 解决方案

使用同批次新电池更换，不混用新旧电池。

### 仍未解决？

更换新电池后仍提示低电量，联系售后检查电池仓触点。

## VIDEO_LOW_BATTERY：可视模块低电量

### 触发条件

小米全自动智能门锁 Pro 的可视门铃或摄像头模块电量不足。

### 排查步骤

1. 在米家 APP 中查看可视模块电量。
2. 按说明取下或连接对应充电接口。
3. 充电完成后重新测试门铃和视频功能。

### 解决方案

及时给可视模块充电。门锁主体和可视模块可能是不同供电单元，应分别关注。

### 仍未解决？

充电后电量不增长或视频功能仍不可用，联系售后检测电池或可视模块。

## VERIFY_FAILED / LOCK_TEMP_LOCKED：验证失败或临时锁定

### 触发条件

指纹、密码、NFC 连续验证失败，或输错次数过多触发安全锁定。

### 排查步骤

1. 停止连续尝试，等待临时锁定结束。
2. 使用其他已授权方式开门。
3. 检查用户权限是否被删除或过期。
4. 对指纹识别失败用户重新录入指纹。

### 解决方案

确认使用有效权限。访客临时密码过期后应重新创建。

### 仍未解决？

所有授权方式均失败时，使用本人机械钥匙开门，并联系售后处理。

## DOOR_AJAR / BOLT_BLOCKED：门虚掩或锁舌受阻

### 触发条件

门未完全关紧，锁舌无法伸出，或门扣板位置偏移导致上锁失败。

### 排查步骤

1. 重新关门，确认门体贴合门框。
2. 检查门缝是否有异物。
3. 开门状态下测试锁舌能否正常伸缩。
4. 若开门状态正常，联系安装服务调整门扣板。

### 解决方案

清除异物，调整门体和门扣板。全自动锁体遇阻时不要反复强行上锁。

### 仍未解决？

开门状态下锁舌也异常，转人工检测锁体或电机。

## PRY_ALARM：防撬报警

### 触发条件

门锁检测到面板被撬动、异常震动或疑似非法拆卸。

### 排查步骤

1. 确认现场安全。
2. 检查门锁面板是否松动。
3. 查看 APP 异常记录。
4. 若近期有维修或安装调整，确认面板是否固定到位。

### 解决方案

面板松动时联系安装或售后重新固定。确认非本人操作的异常报警时，应及时检查门锁和门体安全。

### 仍未解决？

无外力情况下频繁防撬报警，联系售后检测防撬开关。

## NETWORK_OFFLINE：门锁离线或状态不同步

### 触发条件

门锁无法通过蓝牙、网关或家庭中枢同步状态，APP 显示离线或通知延迟。

### 排查步骤

1. 靠近门锁打开米家 APP。
2. 检查手机蓝牙和米家权限。
3. 检查网关或家庭中枢是否在线。
4. 检查门锁电量。

### 解决方案

恢复蓝牙、网关和网络连接。配网和频段问题见 [NET_GUIDE_002] 5G/2.4G WiFi 配网说明（避坑）。

### 仍未解决？

网关正常但门锁长期离线，联系售后检测通信模块。

## SENSOR_ERROR / MOTOR_ERROR：传感器或电机异常

### 触发条件

门锁传感器、锁舌检测、全自动电机或相关控制模块异常。

### 排查步骤

1. 检查门锁电量。
2. 开门状态下测试上锁和解锁动作。
3. 检查门缝和门扣板是否阻碍锁舌。
4. 重启或断电恢复后再次测试。

### 解决方案

若由门体阻力导致，先调整门体和门扣板。若为内部模块异常，不建议自行拆修。

### 仍未解决？

反复提示传感器或电机异常时，停止强行开合锁体并联系售后。
