# 固件回退操作指南（特殊审批）

## 错误码总览

固件回退不是普通用户自助功能。多数米家设备、小爱音箱、门锁、摄像机、空气净化器、扫地机不提供用户自行选择旧版本的入口。只有在升级后出现严重兼容问题、功能不可用或官方确认版本缺陷时，才可能通过特殊审批、售后工具或服务端灰度处理。

| 错误码 | 含义 | 严重程度 | 自助可解 |
|---|---|---|---|
| FW_ROLLBACK_UNAVAILABLE | 当前设备不支持用户自助回退 | 中 | 否 |
| FW_ROLLBACK_REQUEST | 用户申请固件回退 | 中 | 部分可解 |
| FW_ROLLBACK_APPROVED | 回退申请已通过 | 中 | 部分可解 |
| FW_ROLLBACK_REJECTED | 回退申请未通过 | 低 | 否 |
| FW_ROLLBACK_IN_PROGRESS | 回退处理中 | 中 | 否 |
| FW_ROLLBACK_FAILED | 回退失败 | 高 | 否 |
| FW_VERSION_LOCKED | 设备版本被锁定或灰度控制 | 中 | 否 |

## FW_ROLLBACK_UNAVAILABLE：当前设备不支持用户自助回退

### 触发条件

用户在米家 APP 中找不到回退入口，或客服确认设备没有用户侧固件回退功能。

### 排查步骤

1. 进入设备页 > 固件升级。
2. 查看是否有“回退版本”或“恢复旧版本”入口。
3. 阅读升级说明和版本日志。
4. 确认问题是否可通过重启、重新配网或设置恢复解决。

### 解决方案

多数设备不支持自行回退。应先按 [CODE_FW_001] 固件升级操作与失败错误码处理排查当前问题。

### 仍未解决？

升级后严重影响核心功能时，联系人工登记回退申请。

## FW_ROLLBACK_REQUEST：用户申请固件回退

### 触发条件

升级后出现严重问题，用户希望退回旧版本。

### 排查步骤

1. 记录设备型号、当前固件版本和升级时间。
2. 记录故障现象、出现频率和复现步骤。
3. 提供米家 APP 截图、错误提示和设备日志入口。
4. 说明是否已尝试重启、重新配网、恢复设置。

### 解决方案

客服会根据设备型号、版本策略和故障严重程度判断是否提交特殊审批。

### 仍未解决？

资料不足时，回退申请可能无法进入审批，应补充复现视频和日志。

## FW_ROLLBACK_APPROVED：回退申请已通过

### 触发条件

售后或技术支持确认可对指定设备执行回退。

### 排查步骤

1. 保持设备在线、电量充足或接入电源。
2. 保持手机米家 APP 在线。
3. 按客服指定时间进入固件页面。
4. 不要在回退期间断电、重置或删除设备。

### 解决方案

按客服指引完成回退。回退完成后检查设备核心功能是否恢复。

### 仍未解决？

回退后问题仍存在，继续保留日志并转售后检测硬件或账号环境。

## FW_ROLLBACK_REJECTED：回退申请未通过

### 触发条件

客服或系统判断该设备不支持回退，或当前问题不符合回退条件。

### 排查步骤

1. 查看拒绝原因。
2. 确认是否有新版修复计划。
3. 尝试恢复默认设置、重新配网或调整配置。
4. 对影响使用的问题提交故障工单。

### 解决方案

无法回退时，应等待新版本修复或按售后流程处理。

### 仍未解决？

若设备核心功能不可用，应按保修维修流程处理，而不是继续尝试第三方刷机。

## FW_ROLLBACK_IN_PROGRESS：回退处理中

### 触发条件

设备正在执行版本回退或服务端正在下发旧版本。

### 排查步骤

1. 保持网络稳定。
2. 不要退出账号、删除设备或恢复出厂。
3. 等待客服通知或 APP 页面完成。
4. 记录回退开始和结束时间。

### 解决方案

回退期间以等待为主，不进行额外操作。

### 仍未解决？

超过客服给定时间仍未完成，应提供设备在线状态和截图。

## FW_ROLLBACK_FAILED：回退失败

### 触发条件

旧版本下发失败、安装失败、设备离线或回退后仍无法启动。

### 排查步骤

1. 保持设备通电。
2. 不要反复断电或重置。
3. 记录失败提示、指示灯状态和时间。
4. 联系客服继续处理。

### 解决方案

回退失败属于高风险状态，不建议用户继续自行操作。

### 仍未解决？

设备无法恢复时，走维修检测或换机流程，由售后判断。

## FW_VERSION_LOCKED：设备版本被锁定或灰度控制

### 触发条件

设备无法升级或回退到指定版本，提示版本不可用、灰度未开放或版本锁定。

### 排查步骤

1. 确认设备型号、地区和账号。
2. 检查是否为灰度批次。
3. 等待版本开放或联系客服核对。
4. 不要安装非官方固件。

### 解决方案

版本锁定由服务端策略控制，用户侧通常无法绕过。

### 仍未解决？

如版本锁定导致核心功能不可用，提交售后工单并等待官方处理。
