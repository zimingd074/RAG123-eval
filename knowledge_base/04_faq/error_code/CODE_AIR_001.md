# 米家空气净化器错误码与指示灯说明

## 错误码总览

米家空气净化器更常见的是屏幕图标、指示灯颜色、APP 提示和运行状态异常。本手册适用于米家空气净化器 4 Lite、米家空气净化器 4、米家空气净化器 4 Pro。

| 错误码 | 含义 | 严重程度 | 自助可解 |
|---|---|---|---|
| AIR_FILTER_EXPIRED | 滤芯寿命到期或接近到期 | 中 | 是 |
| AIR_FILTER_MISSING | 滤芯未安装或仓盖未闭合 | 中 | 是 |
| AIR_SENSOR_DIRTY | 颗粒物传感器脏污或读数异常 | 中 | 是 |
| AIR_PM_RED | 空气质量灯持续红色 | 低 | 是 |
| AIR_WATER_RISK | 设备受潮或进水风险 | 高 | 部分可解 |
| AIR_FAN_BLOCKED | 风道堵塞或风机异常 | 高 | 部分可解 |
| AIR_NETWORK_OFFLINE | APP 离线或无法配网 | 中 | 是 |
| AIR_DISPLAY_ERROR | 屏幕或触控异常 | 中 | 部分可解 |

## AIR_FILTER_EXPIRED：滤芯寿命到期或接近到期

### 触发条件

米家 APP 或机身屏幕提示滤芯寿命不足、滤芯到期或需要更换。

### 排查步骤

1. 打开米家 APP 查看滤芯剩余寿命。
2. 打开滤芯仓检查滤芯是否有灰尘、异味或受潮。
3. 确认滤芯型号与净化器型号匹配。
4. 更换新滤芯后在 APP 中按提示重置滤芯寿命。

### 解决方案

更换对应型号滤芯。4 Lite、4、4 Pro 滤芯尺寸不同，不应混用。

### 仍未解决？

新滤芯安装后仍提示到期，检查滤芯是否安装到位；仍异常则联系售后检测识别模块。

## AIR_FILTER_MISSING：滤芯未安装或仓盖未闭合

### 触发条件

机器无法启动，提示滤芯异常、滤芯仓打开或仓盖未关闭。

### 排查步骤

1. 断电后打开滤芯仓。
2. 确认滤芯塑封已取下。
3. 按方向重新放入滤芯。
4. 关闭仓盖并确认卡扣到位。
5. 重新通电开机。

### 解决方案

重新安装滤芯和仓盖。滤芯没有装到位时，机器会进入保护状态。

### 仍未解决？

仓盖闭合正常仍提示异常时，联系售后检测仓盖开关或滤芯识别结构。

## AIR_SENSOR_DIRTY / AIR_PM_RED：传感器脏污或红灯持续

### 触发条件

PM2.5 数值异常偏高，空气质量灯持续红色，或空气质量变化与实际环境明显不符。

### 排查步骤

1. 将机器移到开阔位置，远离厨房、香薰、加湿器和烟雾。
2. 短时通风后观察 PM2.5 是否下降。
3. 清理颗粒物传感器进风口周围灰尘。
4. 检查滤芯寿命和是否受潮。
5. 重启设备后观察 10-15 分钟。

### 解决方案

远离污染源、清洁传感器进风区域并更换到期滤芯。

### 仍未解决？

空气质量良好的房间仍长期红灯，可能是传感器异常，应联系售后。

## AIR_WATER_RISK：设备受潮或进水风险

### 触发条件

机器靠近水源、加湿器、窗边雨水，或机身内部可能受潮。

### 排查步骤

1. 立即断电。
2. 移到干燥通风位置。
3. 检查滤芯是否受潮。
4. 滤芯受潮或有霉味时更换新滤芯。
5. 机身完全干燥前不要通电。

### 解决方案

保持设备和滤芯干燥。空气净化器不适合放在浴室、湿衣房或加湿器正旁边。

### 仍未解决？

通电后有焦味、异常声响或无法开机，应停止使用并联系售后。

## AIR_FAN_BLOCKED：风道堵塞或风机异常

### 触发条件

出风量明显变小、噪音变大、出现异响、震动或风机不转。

### 排查步骤

1. 检查顶部出风口和四周进风口是否被遮挡。
2. 断电后检查滤芯仓内是否有塑料袋、纸片、毛发团。
3. 确认滤芯安装方向正确。
4. 重新开机，在低档和高档之间切换观察。

### 解决方案

移开遮挡物，清理可见异物并重新安装滤芯。

### 仍未解决？

异响持续、伴随焦味或风机不转时，应联系售后检测风机。

## AIR_NETWORK_OFFLINE：APP 离线或无法配网

### 触发条件

米家 APP 显示设备离线、无法添加设备或控制延迟。

### 排查步骤

1. 确认手机连接 2.4GHz WiFi。
2. 开启蓝牙、定位和米家 APP 权限。
3. 重启路由器和空气净化器。
4. 按设备说明重置 WiFi 后重新添加。

### 解决方案

使用 2.4GHz WiFi 并关闭路由器 AP 隔离、MAC 过滤。更多步骤见 [FAQ_NET_001] 设备配网失败通用排查指南。

### 仍未解决？

同一网络其他设备正常，该净化器长期离线，应联系售后检测 WiFi 模块。

## AIR_DISPLAY_ERROR：屏幕或触控异常

### 触发条件

屏幕不亮、显示残缺、触控无响应或按键不工作。

### 排查步骤

1. 检查电源是否插紧。
2. 断电 10 秒后重新通电。
3. 在米家 APP 中尝试控制设备。
4. 检查是否开启屏幕熄灭或睡眠模式。

### 解决方案

若 APP 可控制但屏幕不亮，先检查屏幕亮度或熄屏设置。

### 仍未解决？

屏幕、触控和 APP 控制均异常时，联系售后检测控制板。
