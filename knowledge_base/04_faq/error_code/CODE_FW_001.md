# 固件升级操作与失败错误码处理

## 小米手表 S3 系统升级

本手册覆盖两类固件升级问题：一类是用户询问“小米手表 S3 怎么升级系统”“怎么系统更新”等基础操作；另一类是升级失败、卡进度、升级后异常等错误码排查。用户常说的“升级系统”“系统更新”和“固件升级”在客服语境中含义一致。

小米手表 S3 以及同系列小米手表应通过手机端「小米运动健康 / Mi Fitness」App 安装官方固件，不需要电脑，也不建议使用第三方刷机包。

### 升级前准备

1. 手表电量保持在 50% 以上，建议放在原装充电底座上。
2. 手机电量保持在 50% 以上，手机和手表保持蓝牙连接并放在近距离。
3. 手机网络稳定，优先使用 WiFi 下载固件包。
4. 确认手机端「小米运动健康 / Mi Fitness」App 已更新到最新版本。
5. 升级前暂停运动记录、录音或其他持续运行任务，让手表处于空闲状态。

### 升级步骤

1. 打开手机端「小米运动健康 / Mi Fitness」App。
2. 进入小米手表 S3 的设备页。
3. 点击「固件升级」「系统更新」或相近名称的版本更新入口。
4. App 检测到新版本后，按提示下载固件包并同步到手表。
5. 同步完成后，手表会进入升级状态，屏幕可能显示进度条、重启动画或短暂黑屏。
6. 升级期间不要断开蓝牙、关闭 App、切换网络、重启手机或频繁操作手表。
7. 升级完成后手表会自动重启，可在手表「设置 > 关于 > 版本号」确认是否已更新。

单次升级通常需要 10-20 分钟。若进度短时间停留不动，先保持充电和连接，等待设备自动恢复，不要立即强制重启。

### 升级后续航变差

升级完成后的 1-2 天内，手表可能因同步数据、重建索引、表盘组件刷新或健康监测策略变化出现耗电变快。可先观察一个完整充电周期，并检查常亮显示、全天血氧、高频心率、运动自动识别、通知同步等高耗电设置。

如果升级后持续发热、无法充电、反复重启、版本显示异常，或观察 2 个完整充电周期后续航仍明显低于升级前，应转人工核对当前版本、推送批次和设备状态。

## 错误码总览

以下错误码适用于小米手表、米家 APP、小爱音箱、智能门锁、空气净化器、摄像机、智能插座、扫地机器人等设备的固件升级失败场景。不同设备展示的原始错误码可能不同，客服知识库按统一错误类型归类。

| 错误码 | 含义 | 严重程度 | 自助可解 |
|---|---|---|---|
| FW_NO_UPDATE | 未检测到新版本 | 低 | 是 |
| FW_DOWNLOAD_FAIL | 固件包下载失败 | 中 | 是 |
| FW_LOW_BATTERY | 电量不足无法升级 | 中 | 是 |
| FW_NETWORK_LOST | 升级中网络中断 | 中 | 是 |
| FW_INSTALL_FAIL | 安装失败或校验失败 | 高 | 部分可解 |
| FW_DEVICE_BUSY | 设备正在运行，无法升级 | 中 | 是 |
| FW_TIMEOUT | 升级超时 | 中 | 部分可解 |
| FW_BRICK_RISK | 升级后无法启动或离线 | 高 | 否 |

## FW_NO_UPDATE：未检测到新版本

### 触发条件

用户进入固件升级页面，系统提示当前已是最新版本或未检测到新版本。

### 排查步骤

1. 确认设备已在线。
2. 下拉刷新设备页面。
3. 检查米家 APP 是否为最新版本。
4. 确认设备所在地区和账号区域是否正确。

### 解决方案

未检测到新版本通常不是故障。固件推送可能分批进行，不同地区和设备批次时间不同。

### 仍未解决？

若官方已明确该设备应有新版本，但页面长期无更新，联系人工核对设备型号和地区。

## FW_DOWNLOAD_FAIL：固件包下载失败

### 触发条件

升级页面提示下载失败、下载中断或进度一直为 0%。

### 排查步骤

1. 检查手机网络是否正常。
2. 切换到稳定 WiFi。
3. 关闭 VPN、代理或网络加速器。
4. 重启米家 APP 后重新进入升级页面。
5. 稍后避开网络高峰再试。

### 解决方案

使用稳定网络重新下载固件包。下载失败不应影响设备当前版本运行。

### 仍未解决？

多次下载失败且其他网络正常，联系人工核对服务端版本和设备地区。

## FW_LOW_BATTERY：电量不足无法升级

### 触发条件

设备或手机电量低，系统提示无法开始升级。

### 排查步骤

1. 给手机充电至 50% 以上。
2. 给手表、耳机、门铃、门锁可视模块等设备充电。
3. 插电设备确认电源稳定。
4. 重新打开升级页面。

### 解决方案

升级前保证设备和手机电量充足。电池供电设备不建议在低电量状态升级。

### 仍未解决？

充电后仍提示低电量，检查设备电池状态或联系售后检测。

## FW_NETWORK_LOST / FW_TIMEOUT：网络中断或升级超时

### 触发条件

升级进度卡住、提示网络中断、设备离线或升级超时。

### 排查步骤

1. 保持手机、设备和路由器距离较近。
2. 不要在升级期间断电、重启路由器或关闭 APP。
3. 等待 10-20 分钟，观察设备是否自动恢复。
4. 若设备恢复在线，重新检查版本。
5. 若设备仍离线，按设备说明重启后查看。

### 解决方案

网络波动时先等待设备自动恢复，再决定是否重新升级。不要频繁断电。

### 仍未解决？

设备长时间离线或无法进入配网状态，应联系售后，避免反复重置。

## FW_INSTALL_FAIL：安装失败或校验失败

### 触发条件

固件包已下载，但安装阶段失败、校验失败或提示版本不匹配。

### 排查步骤

1. 重启米家 APP。
2. 重启设备。
3. 确认设备型号与固件页面一致。
4. 再次尝试升级。
5. 不要使用非官方固件包。

### 解决方案

官方渠道升级失败时，通常可等待下一次推送或联系客服核对版本。

### 仍未解决？

多次安装失败、版本显示异常或设备功能异常，应转人工处理。

## FW_DEVICE_BUSY：设备正在运行，无法升级

### 触发条件

扫地机正在清扫、净化器正在运行高负载任务、摄像机正在录像或门锁正在执行关键动作。

### 排查步骤

1. 停止设备当前任务。
2. 将扫地机回充到基站。
3. 关闭高负载运行模式。
4. 等待设备空闲后重新升级。

### 解决方案

升级前让设备处于待机、充电或空闲状态。

### 仍未解决？

设备显示空闲但仍提示 busy，重启设备后重试。

## FW_BRICK_RISK：升级后无法启动或离线

### 触发条件

升级后设备无法开机、无法联网、指示灯异常或功能不可用。

### 排查步骤

1. 保持设备通电或电量充足。
2. 等待 10-20 分钟，避免立即断电。
3. 记录错误提示、指示灯状态和升级时间。
4. 不要自行拆机或刷入第三方固件。

### 解决方案

该类问题不建议自助继续操作，应转人工售后处理。

### 仍未解决？

设备升级后无法启动、无法配网或无法恢复出厂时，联系售后并提供订单号、设备型号和升级时间。
