## E01：LDS 激光测距传感器异常

### 触发条件

LDS 激光头被遮挡、卡住，或激光测距模块无法正常旋转。

### 排查步骤

1. 检查 LDS 模组周围是否有异物。
2. 轻轻转动激光头外部结构，确认没有卡滞。
3. 擦拭 LDS 模组外壳。
4. 重启主机后重新启动清扫。

### 解决方案

若清理后仍反复出现 E01，应停止使用并联系客服检测 LDS 模组。

### 仍未解决？

LDS 不旋转、伴随异响或反复报 E01 时，转人工售后。

## E02：碰撞传感器卡住

### 触发条件

前保险杠被家具、线缆或异物卡住，无法正常回弹。

### 排查步骤

1. 检查前保险杠左右两侧是否被异物卡住。
2. 轻按保险杠数次，确认可以正常回弹。
3. 将主机移到空旷地面重新启动。

### 解决方案

清理卡住的异物，并在地图中为固定卡困位置设置禁区。

### 仍未解决？

保险杠无法回弹或清理后仍持续报错时，转人工检测碰撞传感器。

## E03 / E04：轮子悬空或悬崖传感器异常

### 触发条件

主机被搬起、卡在门槛或深色地面导致悬崖传感器误判。

### 排查步骤

1. 将主机放到平整地面。
2. 擦拭底部悬崖传感器。
3. 检查传感器附近是否有灰尘、毛发或水渍。
4. 避免在高落差边缘直接启动。

### 解决方案

清洁传感器并从平整地面重新启动。深色地毯或深色地砖反复误判时，设置禁区。

### 仍未解决？

传感器清洁后仍频繁误判悬崖，转人工检测传感器。

## E05 / E06 / E07：主刷、边刷或主轮卡住

### 触发条件

毛发、线缆、塑料袋、布条等缠绕主刷、边刷或主轮。

### 排查步骤

1. 关闭主机并翻转。
2. 拆下主刷和边刷。
3. 清理刷体、轴承和轮组上的毛发、线缆。
4. 检查刷体和轮组是否能自由转动。
5. 装回后重新启动。

### 解决方案

清理缠绕异物。宠物家庭应提高主刷和边刷清理频率，详见 [MANUAL_VAC_003] 石头扫地机保养维护手册。

### 仍未解决？

刷体和轮组清洁后仍无法转动或伴随异响，转人工检测驱动模块。

## E08 / E24：机器人被困或禁区限制

### 触发条件

机器人进入狭窄区域、被椅脚困住，或任务起点位于禁区、虚拟墙、禁拖区附近。

### 排查步骤

1. 将机器人移到空旷区域。
2. 检查 APP 地图中的禁区、虚拟墙和禁拖区。
3. 清理周围地面线缆、玩具和轻薄地毯。
4. 从充电座或基站重新发起任务。

### 解决方案

修改地图禁区，或为固定卡困点新增禁区。

### 仍未解决？

空旷地面仍原地打转、路线异常或反复被困，转人工检测传感器和轮组。

## E09 / E10：尘盒滤网异常

### 触发条件

尘盒、滤网未安装到位，或滤网堵塞、潮湿。

### 排查步骤

1. 取出尘盒，确认尘盒和滤网是否安装。
2. 清理尘盒内灰尘。
3. 轻拍滤网去除浮尘。
4. 若滤网水洗过，确认完全晾干后再装回。

### 解决方案

重新安装尘盒和滤网。滤网堵塞严重时更换新滤网。

### 仍未解决？

尘盒和滤网安装正确但仍报错，转人工检测尘盒识别开关。

## E11 / E12 / E13 / E14：磁场、电量、充电或电池异常

### 触发条件

主机靠近强磁区域、电量过低、充电触点脏污，或电池保护触发。

### 排查步骤

1. 将主机移离磁条、强磁门吸或强磁物体。
2. 放回充电座或基站充电。
3. 擦拭主机和充电座触点。
4. 检查电源线和插座是否正常。
5. 充电 30 分钟后再次开机。

### 解决方案

清理触点、恢复充电并移离强磁区域。E14 电池异常通常不建议继续自助处理。

### 仍未解决？

反复出现 E13、E14，或充电时异常发热，应停止使用并转人工售后。

## E15 / E16 / E21 / E22 / E23：定位与回充相关异常

### 触发条件

沿墙传感器、回充传感器、竖向碰撞传感器脏污，或充电座/基站信号区域被遮挡。

### 排查步骤

1. 用干布擦拭沿墙传感器和回充传感器。
2. 清理充电座或基站信号发射区域。
3. 将基站靠墙摆放，前方保持无遮挡。
4. 将主机放到基站前方 1 米左右，点击回充。

### 解决方案

清洁传感器和基站信号区域，调整基站摆放位置。

### 仍未解决？

基站位置正确但仍无法识别或无法回充，转人工检测回充传感器。

## E18：风机异常

### 触发条件

风机堵塞、进风口堵住，或风机模块异常。

### 排查步骤

1. 清空尘盒。
2. 检查滤网是否堵塞或潮湿。
3. 检查主刷、风道入口是否有异物。
4. 重启主机后使用标准吸力测试。

### 解决方案

清理尘盒、滤网、主刷和风道。滤网潮湿时完全晾干。

### 仍未解决？

清理后仍报 E18、吸力明显异常或有尖锐异响，转人工检测风机。

## DOCK_WATER / DOCK_WASH / DOCK_DUST：基站提示

### 触发条件

G10 / G10S Pro 基站可能提示清水箱缺水、污水箱已满、清洁盘堵塞、集尘袋未装好或集尘通道堵塞。

### 排查步骤

1. 检查清水箱是否有水。
2. 倒掉污水箱污水并冲洗。
3. 清理基站清洁盘和滤网。
4. G10S Pro 检查集尘袋是否装好或已满。
5. 重新放回主机并启动基站维护任务。

### 解决方案

补水、倒污水、清理清洁盘或更换集尘袋。

### 仍未解决？

基站无抽水声、无集尘声、持续漏水或反复提示基站异常时，转人工售后。
