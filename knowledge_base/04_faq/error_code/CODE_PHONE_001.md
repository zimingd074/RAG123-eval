# 小米手机充电异常排查（含发热警告）

## 错误码总览

小米和 Redmi 手机在充电场景中更常见的是系统提示、充电动画异常或用户可见现象，而非固定数字错误码。本手册按客服问答高频现象组织，适用于小米 13、小米 13 Pro、小米 14、小米 14 Pro、Redmi Note 13、Redmi K70 等 USB Type-C 机型。

| 错误码 | 含义 | 严重程度 | 自助可解 |
|---|---|---|---|
| PHONE_NO_CHARGE | 插上充电器没有反应 | 高 | 部分可解 |
| PHONE_SLOW_CHARGE | 充电很慢或不显示快充 | 中 | 是 |
| PHONE_CHARGE_INTERRUPTS | 充电反复断开 | 中 | 部分可解 |
| PHONE_OVERHEAT_WARNING | 充电时温度过高或提示过热 | 高 | 部分可解 |
| PHONE_MOISTURE_RISK | 充电口潮湿或进液风险 | 高 | 部分可解 |
| PHONE_WIRELESS_FAIL | 无线充电无法启动或速度低 | 中 | 是 |
| PHONE_BATTERY_NOT_FULL | 高温环境下充不满或卡在某个电量 | 中 | 是 |
| PHONE_BATTERY_SWELL | 电池鼓包、异味、异常变形 | 高 | 否 |

## PHONE_NO_CHARGE：插上充电器没有反应

### 触发条件

插入充电器后，手机没有充电动画，电量不增加，或关机状态下无法点亮充电图标。

### 排查步骤

1. 更换墙插或插排，排除插座无电。
2. 更换原装或正规品牌充电器和 USB Type-C 数据线。
3. 检查充电口是否有灰尘、棉絮、变形或液体痕迹。
4. 长按电源键 10 秒尝试强制重启。
5. 若手机已完全没电，连接原装充电器静置 15-30 分钟后再尝试开机。

### 解决方案

若更换插座、充电器和线材后恢复，问题通常来自外部供电或线材。若所有组合均无反应，应停止反复插拔，避免扩大接口损伤。

### 仍未解决？

充电口变形、进液痕迹明显、关机静置充电后仍无图标，建议转人工检测充电接口、电池和主板供电线路。

## PHONE_SLOW_CHARGE：充电很慢或不显示快充

### 触发条件

手机可以充电，但不显示“快充”“超级快充”或“HyperCharge”，充电时间明显长于标称。

### 排查步骤

1. 确认机型标称功率：Redmi Note 13 为 33W，小米 13 为 67W，小米 14 为 90W，小米 13 Pro / 小米 14 Pro / Redmi K70 为 120W。
2. 使用随机附带或对应功率的小米原装充电器和线材。
3. 查看电量区间，接近满电时系统会自动降低充电功率。
4. 暂停游戏、导航、热点、录像等高负载任务。
5. 取下厚保护壳，保持手机背面和充电器通风。

### 解决方案

快充需要手机、充电器、数据线和温度条件共同满足。更换为原装套装并降低负载后，通常可恢复较高充电功率。

### 仍未解决？

原装套装仍无法快充，且不同线材测试结果一致时，建议联系售后检测充电线识别、接口触点和电池状态。

## PHONE_CHARGE_INTERRUPTS：充电反复断开

### 触发条件

插上后充电动画反复出现和消失，或手机在充电与未充电之间来回切换。

### 排查步骤

1. 轻轻晃动数据线接头，观察是否因接触不良断开。
2. 更换 USB Type-C 数据线测试。
3. 清理充电口可见灰尘，不使用金属尖锐物深入刮擦接口。
4. 检查手机是否套了过紧保护壳，导致插头无法完全插入。
5. 重启手机后再次测试。

### 解决方案

线材松动、接口积灰和保护壳遮挡是高频原因。更换线材、清理外部灰尘、取下保护壳后可再次确认。

### 仍未解决？

接口松动、插头无法稳固插入或轻碰即断开，建议转人工检测充电口，不建议继续边充边用。

## PHONE_OVERHEAT_WARNING：充电时温度过高或提示过热

### 触发条件

充电时手机明显烫手，系统提示温度过高，或快充速度自动下降。

### 排查步骤

1. 立即停止游戏、录像、热点、导航和大型下载。
2. 拔下充电器，将手机放到阴凉通风处自然降温。
3. 取下厚保护壳，避免放在被褥、沙发、车内或阳光直射位置。
4. 降温后使用原装充电器重新充电。
5. 在设置 > 电池中查看电池温度或电池状态，若系统提供该入口。

### 解决方案

快充本身会产生热量，高温环境和高负载任务会叠加发热。系统可能主动限速或停止高功率充电，这是保护电池和主板的正常策略。

### 仍未解决？

未运行高负载应用、使用原装充电器仍频繁过热，或出现关机、异味、冒烟、屏幕鼓起，应立即停止充电并联系售后。

## PHONE_MOISTURE_RISK：充电口潮湿或进液风险

### 触发条件

手机淋雨、进水、从潮湿环境拿出后尝试充电，或充电口可见水迹。

### 排查步骤

1. 立即停止充电并拔下线材。
2. 关闭手机或保持待机，不要反复插拔充电器。
3. 用干燥软布擦拭机身外部水迹。
4. 将充电口朝下轻轻放置在通风干燥环境中自然晾干。
5. 晾干前不要使用热风、烘干机、酒精或棉签深入接口。

### 解决方案

即使 IP54 或 IP68 机型，也不建议在潮湿状态下充电。确认接口完全干燥后，再使用原装充电器短时间测试。

### 仍未解决？

进液后无法充电、屏幕异常、扬声器杂音或摄像头起雾，建议转人工检测。人为进液损坏通常不按防水测试条件覆盖。

## PHONE_WIRELESS_FAIL：无线充电无法启动或速度低

### 触发条件

支持无线充电的机型放到无线充电器上无反应，或无线充电速度明显偏低。

### 排查步骤

1. 确认机型支持无线充电：小米 13、小米 13 Pro、小米 14、小米 14 Pro 支持；Redmi Note 13 和 Redmi K70 不支持。
2. 将手机中心与无线充电器线圈区域对齐。
3. 取下厚保护壳、金属支架、磁吸环位置不匹配的配件。
4. 使用无线充电器原配电源适配器。
5. 检查充电器风扇、指示灯和供电是否正常。

### 解决方案

无线充电对位置、保护壳厚度和供电适配器更敏感。对齐线圈、取下干扰配件并使用原配供电后，再观察充电动画和温度。

### 仍未解决？

多台支持无线充电的手机均无法充电，优先检查无线充电器；仅单台手机无法充电时，联系售后检测无线充电线圈。

## PHONE_BATTERY_NOT_FULL：高温环境下充不满或卡在某个电量

### 触发条件

手机能充电，但在高温环境下长时间停在某个电量，或到达接近满电前速度明显变慢。

### 排查步骤

1. 查看手机是否处于高温环境，例如车内、阳台、被褥覆盖或边充边玩。
2. 暂停高负载应用并取下保护壳。
3. 放到阴凉通风处等待温度恢复。
4. 重新连接原装充电器继续充电。
5. 检查是否开启电池保护、智能充电或夜间充电保护。

### 解决方案

高温时系统会限制充电电压和功率，可能导致暂时充不满。温度恢复后，通常可继续充电至正常电量。

### 仍未解决？

常温下多次仍无法充满，或电量显示跳变明显，建议转人工检测电池健康和电量计校准状态。

## PHONE_BATTERY_SWELL：电池鼓包、异味、异常变形

### 触发条件

屏幕或后盖翘起，机身鼓起，有刺鼻异味，充电时异常发烫，或机身局部变形。

### 排查步骤

1. 立即停止充电并拔下电源。
2. 不按压屏幕、后盖或电池鼓起区域。
3. 不继续开机、游戏、录像或尝试放电。
4. 将手机放在远离易燃物的通风处。
5. 尽快联系售后处理。

### 解决方案

电池鼓包和异味属于高风险故障，不属于普通自助排查范围。不得自行拆机、刺破、挤压或加热。

### 仍未解决？

出现鼓包、异味、冒烟、进液后发热或无法断电时，立即停止使用并转人工售后；必要时按当地安全指引处理。
