# 石头扫地机常见故障排查（不开机/吸力弱/卡死）

## 故障一：主机不开机

### 可能原因

1. 电池电量耗尽。
2. 主机未正确接触充电座或基站触点。
3. 主机处于低温、高温或保护状态。
4. 电池或主板异常。

### 排查步骤

1. 将主机放回充电座或基站，观察指示灯是否亮起。若亮起，充电 30 分钟后再尝试开机；若不亮，进入步骤 2。
2. 检查充电座或基站电源线是否插紧，插座是否有电。若电源正常，进入步骤 3。
3. 擦拭主机和充电座触点，确认没有灰尘、水渍或氧化痕迹。若仍无法充电，进入步骤 4。
4. 将主机移至 0-40 摄氏度环境静置 30 分钟后再充电。若仍不开机，建议联系客服检测电池或主板。

### 解决方案

- 电量耗尽：放回充电座或基站充电后再启动。
- 触点脏污：断电后使用干布清洁触点。
- 环境温度异常：恢复到常温环境后再使用。

### 仍未解决？

主机充电指示灯始终不亮、充电后仍无法开机、或伴随烧焦味/异常发热时，应停止使用并转人工售后。

## 故障二：吸力变弱

### 可能原因

1. 尘盒已满。
2. 滤网堵塞或未完全晾干。
3. 主刷、边刷、风道被毛发或异物堵住。
4. 吸力档位设置过低。

### 排查步骤

1. 打开尘盒检查灰尘量。若尘盒已满，清空后再测试；若未满，进入步骤 2。
2. 取出滤网检查是否堵塞或潮湿。若滤网潮湿，完全晾干后再装回；若滤网正常，进入步骤 3。
3. 拆下主刷和边刷，清理缠绕毛发。若仍吸力弱，进入步骤 4。
4. 在 APP 中将吸力调至强力或 MAX 档，再运行 3 分钟观察效果。

### 解决方案

- 尘盒满：及时倒尘。
- 滤网堵塞：水洗后完全晾干，或更换新滤网。
- 主刷缠绕：清理主刷两端轴承和刷体毛发。
- 档位过低：切换到强力或 MAX 档。

### 仍未解决？

清理尘盒、滤网、主刷、风道后吸力仍明显偏弱，且主机伴随尖锐异响时，应联系客服检查风机。

## 故障三：机器人频繁卡死或被困

### 可能原因

1. 地面有数据线、袜子、宠物玩具、轻薄地毯。
2. 门槛高度超过 2cm。
3. 地图禁区设置不足。
4. 传感器被灰尘遮挡。

### 排查步骤

1. 查看 APP 中的卡困位置。若多次集中在同一区域，进入步骤 2；若位置随机，进入步骤 4。
2. 检查该区域是否有线缆、地毯边缘、椅脚密集区。若有，先移除或收纳；若没有，进入步骤 3。
3. 在地图中添加禁区、虚拟墙或禁拖区，然后重新清扫。
4. 擦拭主机前部避障传感器、悬崖传感器和 LDS 模组外壳，再重新运行。

### 解决方案

- 线缆和轻薄织物：清扫前收纳。
- 高门槛：设置虚拟墙或手动搬运。
- 固定卡困点：添加禁区。
- 传感器脏污：使用干布清洁。

### 仍未解决？

清理地面和传感器后仍频繁报错、原地打转或撞击家具，应联系售后检测传感器和轮组。

## 故障四：拖地不出水或水量很小

### 可能原因

1. 水箱缺水。
2. 拖布未安装到位。
3. 水量档位设置过低。
4. 出水口或水路堵塞。

### 排查步骤

1. 检查水箱水量。若缺水，补水后再测试；若水量正常，进入步骤 2。
2. 重新安装拖布和拖布支架，确认卡扣完全扣紧。
3. 在 APP 中将水量调高，运行 3-5 分钟观察拖布是否变湿。
4. 若仍不出水，检查出水口是否有水垢或异物。

### 解决方案

- 水箱缺水：加入清水。
- 拖布未装好：重新安装并确认卡扣。
- 水量过低：在 APP 中提高水量。
- 水路疑似堵塞：停止使用非官方清洁剂，并清理出水口。

### 仍未解决？

补水、重装拖布、调高水量后仍完全不出水，应联系客服检查电控水箱或水泵。

## 故障五：G10 / G10S Pro 基站不洗拖布

### 可能原因

1. 清水箱缺水或未安装到位。
2. 污水箱已满或未安装到位。
3. 清洁盘、滤网或水路被毛发和泥污堵塞。
4. 主机未正确停靠在基站。

### 排查步骤

1. 检查清水箱是否有水、污水箱是否已满。若异常，处理后再运行回洗；若正常，进入步骤 2。
2. 取出并重新安装清水箱、污水箱，确认卡扣和密封圈位置正常。
3. 检查基站清洁盘和滤网是否堆积毛发、泥沙。若有，清理后再测试。
4. 让主机退出基站后重新回充，确认停靠位置是否准确。

### 解决方案

- 清水箱缺水：加入清水。
- 污水箱已满：倒掉污水并冲洗污水箱。
- 清洁盘堵塞：拆下清理后装回。
- 停靠偏移：清理基站前方障碍并重新回充。

### 仍未解决？

水箱和清洁盘正常，但基站无抽水声、无回洗动作或持续报水路异常，应联系客服检查基站水泵和传感器。
