# 小米智能门锁常见故障排查（指纹/电池/异常）

## 故障一：指纹识别失败

### 可能原因

1. 手指或识别区有水渍、油污、灰尘。
2. 指纹录入角度单一。
3. 指纹磨损、脱皮或过浅。
4. 门锁电量过低。

### 排查步骤

1. 擦干手指和指纹识别区，再尝试识别。若仍失败，进入步骤 2。
2. 使用密码或 NFC 开门后，进入米家 APP 用户管理。
3. 为同一用户补录常用手指或重新录入指纹。
4. 检查电池电量，低电量时先更换电池或充电。

### 解决方案

- 手指潮湿：擦干后再识别。
- 录入不充分：重新录入多个角度。
- 老人或儿童指纹浅：保留密码、NFC 等备用开锁方式。

### 仍未解决？

多个用户指纹都无法识别、识别区无响应或伴随硬件异常提示时，应联系售后检测指纹模组。

## 故障二：密码无法开锁

### 可能原因

1. 密码输入错误。
2. 临时密码过期。
3. 周期密码不在可用时间段。
4. 连续输错触发临时锁定。

### 排查步骤

1. 确认输入的是当前有效密码。若是临时密码，进入步骤 2。
2. 在米家 APP 中检查临时密码或周期密码的有效期。
3. 等待试错锁定时间结束后再输入。
4. 使用其他已授权方式开门后，重新设置密码。

### 解决方案

- 临时密码过期：重新创建。
- 周期密码时间不匹配：调整有效时间段。
- 试错锁定：等待锁定结束，不要继续反复尝试。

### 仍未解决？

管理员密码和所有备用方式均不可用时，使用本人保管的机械钥匙开门，并联系售后处理。

## 故障三：门锁低电量或没电

### 可能原因

1. 干电池电量耗尽。
2. 可视模块锂电池电量不足。
3. 长期未更换电池。
4. 低温环境导致电池放电能力下降。

### 排查步骤

1. 查看米家 APP 或门锁语音提示是否有低电量提醒。
2. 若仍可开门，进入室内更换干电池或为可视模块充电。
3. 若门外无法电子开锁，使用 Type-C 应急供电口临时供电。
4. 若应急供电无效，使用机械钥匙开门。

### 解决方案

- 低电量提醒出现后及时更换电池。
- 可视门锁需同时关注门锁供电和可视模块供电。
- 机械钥匙应分开放置，避免全部留在室内。

### 仍未解决？

更换新电池后仍频繁提示低电量，可能存在电池仓接触或主板异常，应联系售后。

## 故障四：门关不上或锁舌异常

### 可能原因

1. 门体下沉或变形。
2. 门扣板位置偏移。
3. 锁舌与门框摩擦。
4. 全自动锁体遇阻保护。

### 排查步骤

1. 开门状态下测试锁舌能否正常伸缩。若正常，进入步骤 2。
2. 关门时观察门缝和锁舌是否对准门扣板。
3. 检查门是否需要用力上提或按压才能关严。
4. 若门体位置偏移，联系安装服务调整门扣板或门体。

### 解决方案

- 门扣板偏移：调整门扣板位置。
- 门体下沉：先处理门体合页或门缝问题。
- 全自动锁体遇阻：清除门缝异物，不强行反复上锁。

### 仍未解决？

开门状态下锁舌也无法正常伸缩，或全自动锁体持续异响，应联系售后检测锁体。

## 故障五：APP 显示离线或通知不到

### 可能原因

1. 手机距离门锁较远，蓝牙未连接。
2. 网关离线或家庭中枢不可用。
3. 手机关闭了米家通知权限。
4. 门锁电量不足。

### 排查步骤

1. 靠近门锁，打开米家 APP 查看是否恢复连接。
2. 检查蓝牙、网关和路由器状态。
3. 检查手机系统通知权限和米家 APP 通知设置。
4. 检查门锁电量。

### 解决方案

- 蓝牙离线：靠近门锁后同步状态。
- 网关离线：恢复网关或家庭中枢网络。
- 通知关闭：开启系统通知和米家通知。

### 仍未解决？

同一网关下其他设备正常，门锁长期无法同步状态时，联系售后检测通信模块。
