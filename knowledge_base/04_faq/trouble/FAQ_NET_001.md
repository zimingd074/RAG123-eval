# 设备配网失败通用排查指南

## 故障一：米家 APP 找不到设备

### 可能原因

1. 设备未进入配网模式。
2. 手机蓝牙、定位或本地网络权限未开启。
3. 设备已绑定其他账号。
4. 手机和设备距离太远。

### 排查步骤

1. 查看设备指示灯或语音提示。如果未闪烁或未提示待连接，按说明重置 WiFi；否则进入步骤 2。
2. 打开手机蓝牙、定位和米家 APP 本地网络权限。如果权限正常，进入步骤 3。
3. 将手机靠近设备 1 米内重新搜索。如果仍找不到，进入步骤 4。
4. 检查设备是否已在旧账号中绑定。若已绑定，先用旧账号删除设备。

### 解决方案

- 设备未待连接：按设备说明长按重置键或组合键。
- 权限不足：开启蓝牙、定位、本地网络和附近设备权限。
- 已绑定旧账号：在旧账号中解除绑定后重新添加。

### 仍未解决？

同一手机能添加其他设备，但该设备始终无法被发现，应联系售后检测蓝牙或 WiFi 模块。

## 故障二：输入 WiFi 密码后连接失败

### 可能原因

1. 手机连接了 5GHz WiFi。
2. WiFi 密码输入错误。
3. 路由器开启双频合一、AP 隔离或 MAC 过滤。
4. 公司、校园或酒店网络需要网页认证。

### 排查步骤

1. 确认手机连接 2.4GHz WiFi。如果不是，切换到 2.4GHz 后重试。
2. 重新输入 WiFi 密码，注意大小写和空格。
3. 登录路由器后台，关闭 AP 隔离、MAC 过滤和黑名单。
4. 若使用公司或校园网络，改用家庭路由器或 2.4GHz 手机热点测试。

### 解决方案

- 频段错误：按 [NET_GUIDE_002] 拆分 2.4GHz 和 5GHz。
- 路由限制：关闭新设备接入限制。
- 认证网络：改用普通家庭 WiFi。

### 仍未解决？

同一 WiFi 下多台米家设备都无法配网，优先检查路由器设置；仅单台设备失败时，转人工检测设备网络模块。

## 故障三：配网到 90% 或 99% 后失败

### 可能原因

1. 设备已连上路由器，但绑定云端失败。
2. 手机网络在 WiFi 和移动数据之间切换。
3. 路由器 DNS、IPv6 或防火墙设置异常。
4. 米家服务或账号状态异常。

### 排查步骤

1. 保持手机连接同一个 WiFi，不要切换移动数据。
2. 重启路由器和设备后重新配网。
3. 关闭路由器高级防火墙、儿童上网或设备隔离功能后测试。
4. 退出米家 APP 后重新登录账号。

### 解决方案

- 手机网络切换：配网期间关闭移动数据或保持 WiFi 稳定。
- 云端绑定失败：重启设备和 APP 后重新添加。
- 路由器异常：恢复普通家用网络设置后重试。

### 仍未解决？

多个设备都卡在同一进度，应稍后重试并检查米家服务；只有该设备反复失败，联系售后。

## 故障四：配网成功后很快离线

### 可能原因

1. 设备距离路由器太远。
2. 设备连接到信号弱的 Mesh 节点。
3. 路由器重启后 IP 冲突。
4. 设备电量不足或供电不稳。

### 排查步骤

1. 将设备靠近主路由器，观察是否恢复在线。
2. 重启路由器和设备。
3. 检查设备电源、电池或适配器。
4. 在路由器后台查看设备是否频繁掉线。

### 解决方案

- 信号弱：增加 Mesh/中继或调整路由器位置。
- 供电不稳：更换电源适配器、线材或电池。
- IP 冲突：重启路由器或为设备保留地址。

### 仍未解决？

设备靠近路由器仍持续离线，且其他设备正常，应联系客服检测网络模块或主板。

## 故障五：更换路由器后全部设备离线

### 可能原因

1. WiFi 名称或密码变化。
2. 新路由器关闭 2.4GHz。
3. 新路由器开启双频合一或设备隔离。
4. 设备未重新配网。

### 排查步骤

1. 将新路由器 WiFi 名称和密码改成旧路由器一致。
2. 等待设备自动重连 3-5 分钟。
3. 若仍离线，逐台重置 WiFi 并重新添加。
4. 重新检查自动化和小爱音箱语音控制。

### 解决方案

- 保留旧 WiFi 名称和密码是最低成本迁移方式。
- 名称密码已变化时，按 [NET_GUIDE_003] 路由器更换后设备重连指南重新配网。

### 仍未解决？

新路由器下所有设备都无法连接时，应先用手机热点测试单台设备，区分路由器问题和设备问题。
