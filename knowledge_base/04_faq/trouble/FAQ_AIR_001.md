# 米家空气净化器常见故障排查（不工作/红灯/异响）

## 故障一：空气净化器不工作

### 可能原因

1. 电源未接通。
2. 滤芯仓盖未关闭到位。
3. 滤芯未安装或安装不到位。
4. 设备进入保护状态。

### 排查步骤

1. 检查插座是否有电，电源线或适配器是否插紧。若电源正常，进入步骤 2。
2. 打开滤芯仓，确认滤芯已安装且塑封已取下。若滤芯正常，进入步骤 3。
3. 重新关闭滤芯仓盖，确认卡扣到位。若仍不工作，进入步骤 4。
4. 断电 10 秒后重新通电，再短按电源键开机。

### 解决方案

- 电源异常：更换插座或重新插紧电源线。
- 滤芯仓未闭合：重新关闭滤芯仓盖。
- 滤芯未安装：安装对应型号滤芯。

### 仍未解决？

通电后无任何显示、按键无响应或反复自动关机，应联系客服检测电源板或控制模块。

## 故障二：空气质量灯一直红色

### 可能原因

1. 室内 PM2.5 或粉尘确实较高。
2. 机器摆放位置靠近污染源。
3. 颗粒物传感器进灰。
4. 滤芯寿命不足或堵塞。

### 排查步骤

1. 打开窗户短时通风，观察 PM2.5 数值是否下降。若不下降，进入步骤 2。
2. 检查机器是否靠近厨房、香薰、加湿器、烟雾或灰尘源。若靠近，移至开阔位置。
3. 检查滤芯状态。若滤芯寿命不足或异味明显，更换滤芯。
4. 清理传感器进风区域，重启机器后观察数值。

### 解决方案

- 污染源导致：远离污染源并持续净化。
- 滤芯问题：更换对应型号滤芯。
- 传感器脏污：清理传感器进风区域。

### 仍未解决？

在空气质量良好的房间中仍长期显示红色，且多次重启无效，应联系客服检测颗粒物传感器。

## 故障三：出风有异味

### 可能原因

1. 滤芯吸附饱和。
2. 滤芯受潮产生霉味。
3. 长时间吸附厨房油烟、香薰或宠物异味。
4. 滤芯塑封未取下导致异常气味。

### 排查步骤

1. 关闭机器并打开滤芯仓。
2. 检查滤芯塑封是否已取下。若未取下，取下后重新安装。
3. 闻滤芯是否有霉味、酸味或明显异味。
4. 查看米家 APP 中滤芯寿命。
5. 若滤芯异味明显或寿命不足，更换滤芯。

### 解决方案

- 滤芯未拆封：取下塑封。
- 滤芯受潮或吸附饱和：更换新滤芯。
- 房间污染源明显：配合通风并远离油烟、香薰源。

### 仍未解决？

更换新滤芯后仍有明显焦味、塑料烧灼味或机器内部异味，应停止使用并联系客服。

## 故障四：运行时异响或噪音变大

### 可能原因

1. 出风口或进风口被遮挡。
2. 机身内部进入异物。
3. 滤芯安装不到位。
4. 风机异常。

### 排查步骤

1. 检查机器顶部出风口和四周进风口是否被遮挡。
2. 关闭电源，打开滤芯仓，确认滤芯是否安装到位。
3. 检查滤芯仓内是否有纸片、塑料袋、毛发团等异物。
4. 重新开机，切换低档位和高档位观察噪音变化。

### 解决方案

- 进出风遮挡：移开遮挡物，保证通风空间。
- 滤芯偏位：重新安装滤芯。
- 异物进入：断电后清理可见异物。

### 仍未解决？

异响持续存在、伴随震动或焦味时，应停止使用并联系客服检测风机。

## 故障五：APP 显示离线或无法配网

### 可能原因

1. 手机连接了 5GHz WiFi。
2. 路由器开启双频合一导致设备无法稳定连接。
3. 路由器开启 AP 隔离、MAC 过滤或黑名单。
4. 设备距离路由器太远。

### 排查步骤

1. 确认手机连接 2.4GHz WiFi。
2. 开启手机蓝牙和定位权限。
3. 临时关闭路由器双频合一，拆分 2.4GHz 和 5GHz 名称。
4. 将设备靠近路由器重新配网。
5. 检查路由器是否限制新设备接入。

### 解决方案

- 频段错误：使用 2.4GHz WiFi。
- 权限不足：开启蓝牙、定位和米家 APP 权限。
- 路由限制：关闭 AP 隔离、MAC 过滤或黑名单。

### 仍未解决？

同一 WiFi 下其他米家设备正常，空气净化器多次重置仍无法联网，应联系客服检测 WiFi 模块。
